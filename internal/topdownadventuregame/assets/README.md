# Ebi Relic Dungeon object art

These four original objects were generated with the built-in OpenAI image
generation tool in `stylized-concept` mode on 2026-07-16 and are licensed under
Apache-2.0 with EbiShowcase. No third-party image was used as input.

The source prompt requested exactly four top-down three-quarter-view panels on
a uniform `#ff00ff` background: an orange shrimp adventurer in navy-gold armor
with coral sword, an aqua-gold pearl moon key, a coral-brass cyan-lit treasure
chest, and an obsidian-gold abyssal crab guardian. It required contemporary
premium 2D mobile-game rendering, crisp 48–90px silhouettes, original designs,
and no text, UI, logo, watermark, pixel art, flat vectors, or copyrighted
resemblance. The atlas was chroma-keyed with the image-generation skill helper
(`soft-matte`, edge feather, spill cleanup, despill) and split into four PNGs.

The game decodes these images in `newGame`; `Draw` only selects and transforms
them from existing game state.
