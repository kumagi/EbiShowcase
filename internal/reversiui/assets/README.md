# Reversi championship art

`reversi-championship.png` is an original generated production asset used by
the playable Ebitengine Reversi lessons. It was generated for this repository
on 2026-07-16 and is distributed under Apache-2.0 with the rest of Ebi
Showcase.

Prompt summary: a portrait contemporary premium mobile-game background showing
an underwater pearl-observatory tournament hall, with a quiet dark tabletop,
calm header and information areas, no text, grid, pieces, logo, or watermark.
The source output is preserved in the local Codex generated-image store; this
copy is the runtime asset embedded by `reversiui`.
