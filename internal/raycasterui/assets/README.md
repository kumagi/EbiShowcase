# Raycaster production art

These original generated assets are embedded by `raycasterui` and distributed
under Apache-2.0 with Ebi Showcase.

- `wall-textures.png`: three front-facing underwater-temple materials (pearl
  masonry, teal glass vault, and violet obsidian coral) sampled by the live
  raycaster one vertical wall column at a time.
- `raycast-actors.png`: an abyssal guardian, moon-pearl key, and shell escape
  portal. The source was generated on a flat magenta key and processed with the
  Codex imagegen chroma-key helper (soft matte and despill).

Both source prompts requested contemporary premium mobile-game rendering, no
text, logos, or watermark. Generated for this repository on 2026-07-16.
