## Changes

- Added the pure `audiolab.Gate`: player creation is impossible until explicit keyboard, pointer, or touch input arms it.
- Added `audio-gate`, a playable Go/WASM demo that remains visibly silent until the first gesture, then creates an Ebitengine player using an original synthesized sine beep.
- Added `/ja/labs/audio/` and `/en/labs/audio/` and registered their generator with the build.

## Manual review

- Before activation, the demo shows `TAP / SPACE TO ENABLE SOUND`; after keyboard, pointer, or touch it changes state and starts the original audio player.
- The bilingual pages explain the Gesture → Gate → Player sequence and use no third-party sound asset.
- Silence remains a valid fallback for users/devices that disable audio.
