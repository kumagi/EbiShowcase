# P0-04 — Add a fast local build mode that skips unchanged OGP generation while CI keeps the full build.

Status: PASS

## Required evidence

- [x] Implementation
- [x] Automated checks
- [x] Manual review

## Changes

- Files: `scripts/build.sh`, `scripts/ogp-cache.mjs`, `.gitignore`, `README.md`, and `AGENTS.md`.
- Behavior: `bash scripts/build.sh --fast` still regenerates lesson HTML and reinjects OGP metadata, but skips the expensive OGP PNG renderer only when the normalized generated manifest, renderer sources, bundled font, `go.mod`, and `SITE_ORIGIN` match a verified local cache. Normal builds and any build with `CI` set always render all OGP PNGs.

## Manual checks

- Viewport / input: not applicable; build caching does not change game controls or layout.
- Japanese / English: the fast build regenerated both language trees and the metadata gate passed all bilingual pages.
- Screenshots: `ogp-cache verify` checked all 557 manifest image paths for a valid PNG signature before allowing the skip.
