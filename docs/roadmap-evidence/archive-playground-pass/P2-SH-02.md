## Changes

- Added `palette.kage` and `Palette.Draw`, which passes named `Time` and `Flash` uniforms without coupling game rules to rendering.
- Added the `uniform-palette` playable demo: time continuously changes palette; pointer/touch/Space triggers a decaying flash uniform. The fallback reproduces the same two ideas with ordinary draws.
- Expanded the bilingual Shader Lab hub with both the safe base demo and the uniform demo.
