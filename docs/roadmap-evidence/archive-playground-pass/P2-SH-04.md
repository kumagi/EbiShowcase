## Changes

- Added a `status.kage` effect and optional `Status` helper. A `Damage` display strength separates red/blue samples, while a `Status` value tints poison or freeze after the game has resolved its own rules.
- Added the `status-colors` WASM demo. Tap, mouse, or Space cycles poison, normal, and freeze and triggers a brief damage separation.
- Added the fourth bilingual Shader Lab panel, explicitly teaching that combat data and visual feedback are different layers.
