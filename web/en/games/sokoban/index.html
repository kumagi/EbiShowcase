HOW IT WORKS

Read the meaning of each tile

A stage can be stored as rows of characters: # is a wall, $ is a box, and . is a goal. When the next tile has a box, move the box and player only if the tile beyond is empty.

next := player.Add(direction)
if boxAt(next) && isEmpty(next.Add(direction)) {
    moveBox(next, direction)
}
movePlayer(next)
