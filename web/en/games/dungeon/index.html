HOW IT WORKS

Represent what an enemy is doing as state

Each enemy can be patrolling or chasing. Switch to chase when the player comes close and return to patrol when they escape. Changing Update behavior by state creates an enemy that appears to think.

switch enemy.State {
case Patrol: enemy.walkRoute()
case Chase:  enemy.moveToward(player)
}

GAME + LESSON

Split AI into small decisions

01

Sense

Measure the distance from the enemy to the player.

02

Decide

Use distance to switch between patrol and chase.

03

Act

Run the movement that belongs to the selected state.
