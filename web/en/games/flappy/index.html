DEEP DIVE / ACCELERATION

Why does the bird
fall faster and faster?

The answer is acceleration. It simply means “the amount that changes speed,” like a ball rolling faster down a hill.

1

Position

Where the bird is now. The top is 0, and the number gets bigger as you move down.

birdY = 320
2

Velocity

How far and which way the bird moves each update. Negative goes up; positive goes down.

velocity = -7.4
3

Acceleration

How much velocity changes each time. Keep adding +0.42 as gravity and the bird slows, then falls.

gravity = +0.42

TRY IT / 1 FRAME AT A TIME

Move one frame at a time

A game works like a flipbook. Each step adds gravity to velocity, then moves the bird by that velocity.

“Flap” sets the upward velocity to -7.4, then advances one frame.

THE TWO-LINE RULE

velocity = velocity + gravitythenposition = position + velocity

Order matters. First gravity changes velocity, then velocity changes position. Repeat every frame to make a natural jumping curve.

IN THE REAL GAME

Put it inside
Update

Update reads input, changes velocity, and moves the bird. Draw paints it at the newly calculated position.

const gravity = 0.42
const flapV   = -7.4

func (g *game) Update() error {
    // A press gives upward speed.
    if justPressed() {
        g.velocity = flapV
    }

    // 1. Gravity changes velocity.
    g.velocity += gravity

    // 2. Velocity changes position.
    g.birdY += g.velocity
    return nil
}

WITHOUT ACCELERATION

One constant speed

The bird travels up and down in straight lines, so movement feels mechanical.

WITH ACCELERATION

A smooth arc

Upward velocity weakens, nears zero at the top, then grows downward.

YOUR FIRST TUNING

Experiment with numbers

Larger gravity makes a heavier bird. A more negative flapV makes a stronger flap. Game design is finding values that feel right.
