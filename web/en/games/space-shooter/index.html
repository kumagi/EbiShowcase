HOW IT WORKS

Collect each kind in a slice

Even with one hundred shots on screen, they do not need individual variable names. Add each shot's position and speed to a slice, move every member in Update, then remove shots that leave the screen or hit an enemy.

for i := range bullets {
    bullets[i].Y += bullets[i].VY
}
bullets = removeOutside(bullets)
