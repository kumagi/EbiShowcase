FIRST 30 MINUTES

Your first 30-minute build
on a prepared PC.

Go installation time is not part of the thirty minutes. Start after setup, change one working rule, and make one test pass. If time runs out, finishing the current card is a valid goal for today.

BEFORE

Check the setup

If needed, finish setup first. On a prepared PC, check go version, then start the timer.

00–10

Save a working foundation

Save main.go from Build STEP 01 in your folder and open the empty window with go run ..

10–22

Change the score rule

Replace your current file with the complete main.go from Build STEP 03, then change its addScore to a two-point rule. Confirm SCORE 02 in the window.

22–30

Protect the change with a test

Change the expected value in the STEP 03 test to 4 and run go test ./.... When it is green, leave one comment naming the next rule you want.
