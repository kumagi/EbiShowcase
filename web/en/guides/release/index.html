Five checks before release

01

Local

Run `go test ./...`. Try desktop and phone widths, keyboard, pointer, and touch once each.

02

WASM

Run `GOOS=js GOARCH=wasm go build`. HTML hosts; the game itself is Go/Ebitengine WASM.

03

Pages

Check the Actions artifact and Settings → Pages. Open both `/ja/` and `/en/` after deployment.

04

License

Your code is Apache-2.0. Record original rights for fonts, dependencies, and assets in `THIRD_PARTY_NOTICES.md`.

05

Release

Review title, description, OGP, language links, feedback form, and README instructions.
