STEP 00

Open a place to type commands

You’ll use a terminal—not the browser—to talk to the computer. The name differs by OS.

Windows

  1. Press the Windows key
  2. Type powershell
  3. Open Windows PowerShell or Terminal

macOS

  1. Press Command + Space
  2. Type Terminal
  3. Open the Terminal app

If you see a blinking cursor, you’re ready. You’ll paste commands there.

Name tip: on Windows people often say PowerShell; on Mac, Terminal. Same job—a window where you type commands. Windows 11’s “Terminal” app can host PowerShell. This guide says PowerShell for Windows and Terminal for Mac.

STEP 01

Install Go

Ebitengine is a Go game engine, so install Go first. This repository uses Go 1.25 or later.

Windows

  1. Open https://go.dev/dl/
  2. Download the Microsoft Windows .msi (usually the latest stable at the top)
  3. Run the installer and click through Next
  4. When it finishes, close the PowerShell window and open it again so PATH refreshes

macOS

  1. Open https://go.dev/dl/
  2. Pick the right .pkg: ARM64 for Apple Silicon (M1/M2/M3…), x86-64 for Intel
  3. Run the package and follow the prompts
  4. When it finishes, quit Terminal and open it again

School or work PCs may need an admin password. Ask a parent or teacher if you can’t install.

STEP 02

Mac only: install a C compiler

Ebitengine also uses a little C under the hood. Windows school PCs can skip this. On Mac only, install Apple’s Command Line Tools—the behind-the-scenes tool kit used to build programs. You do not need to memorize the long name. If your classroom PC is Windows, jump to step 3.

On Windows, skip this step and go to “3. Verify”.

xcode-select --install
  1. Paste the command above into Terminal and press Enter
  2. In the dialog, choose Install (it can take several minutes)
  3. If tools are already present, a message like “already installed” is fine

STEP 03

Check that Go works

Type this in the terminal and press Enter:

go version

Success looks like go version go1.25.0 windows/amd64 — any go1.25+ is fine.

If you see “command not found” or “not recognized”: reopen the terminal after install; if it still fails, reboot once.

Optional extra check

Running the official rotate sample proves a window can open. The first run downloads packages.

go run github.com/hajimehoshi/ebiten/v2/examples/rotate@latest

A spinning gopher window means your environment is ready. Close it and continue.

STEP 04

Make a project folder

Desktop or Documents is fine. Here we create ebi-empty under your home folder.

  1. mkdir ebi-emptyCreate the folder
  2. cd ebi-emptyEnter it
  3. go mod init example.com/ebi-emptyCreate the go.mod table of contents

go.mod is the project’s table-of-contents note. It records the module name—the address used to name your code—and lets Go manage borrowed parts such as Ebitengine. With it, Go can prepare the same external packages again. Seeing the small go.mod file means success; you can rename the module later.

STEP 05

Write main.go (do-nothing window)

In a text editor (Notepad, VS Code, Cursor…), create main.go in that folder and paste this exactly:

main.go 
package main

import (
	"image/color"
	"log"

	"github.com/hajimehoshi/ebiten/v2"
)

// Game holds the three methods Ebitengine expects.
type Game struct{}

// Update reads input and advances numbers. State changes belong here.
func (g *Game) Update() error {
	return nil
}

// Draw only projects current state. It never mutates state.
func (g *Game) Draw(screen *ebiten.Image) {
	screen.Fill(color.RGBA{20, 28, 48, 255})
}

// Layout returns the game's internal resolution.
func (g *Game) Layout(outsideWidth, outsideHeight int) (int, int) {
	return 640, 480
}

func main() {
	ebiten.SetWindowSize(640, 480)
	ebiten.SetWindowTitle("Empty Window")
	if err := ebiten.RunGame(&Game{}); err != nil {
		log.Fatal(err)
	}
}
UpdatePer-tick input and state changes. Empty for now.
DrawPer-frame projection. One solid fill; no state changes.
LayoutInternal width and height.
RunGameStarts the loop.

STEP 06

Fetch dependencies and run

Still inside ebi-empty, run these in order:

  1. go mod tidyDownload Ebitengine and friends
  2. go run .Run the program

A dark-blue window titled Empty Window means success. Close it with the window button. Nothing moves yet—but the game foundation is alive.

Troubleshooting

This is where people get stuck most. Match your symptom below. Still stuck? Copy the full red error into the feedback box at the bottom.

PowerShell vs Terminal—which one?

On Windows, PowerShell is fine. On Mac, use Terminal. Same job: a window for typing commands. Windows 11’s Terminal app is OK if it opens PowerShell inside.

go: command not found / not recognized

① After installing Go, fully close and reopen the command window. ② On Windows, check Start for “Go”. ③ Still stuck? Reboot, then try go version again.

xcrun: error: invalid active developer path… (Mac)

Run step 02’s xcode-select --install again. If it stalled, reboot and retry.

Network errors on go mod tidy / go run

Check internet and school/work proxies. Home networks often work when campus ones don’t.

Window flashes and disappears / panic

Read the full red error in the command window. Search it, or paste it into the feedback box below.

Using WSL (Linux on Windows)

Official docs may require GOOS=windows for go run. Prefer plain PowerShell for this first guide.
