SYSTEM GUIDE / PERFORMANCE

Before making it fast,
choose what to measure.

“Slow” is not one problem. Measure per-frame allocation, drawing things off screen, and every-object collision checks separately.

Three small tools

01

Reuse a slice

Pack visible objects into `dst[:0]`. Our Cull benchmark measured 0 allocs/op.

02

Pool

Return short-lived bullets and sparks instead of allocating them every time.

03

Spatial grid

Inspect candidates in nine nearby cells, not every object against every other object.
