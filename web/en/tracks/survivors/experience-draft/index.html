DEEP DIVE / XP AND LEVEL-UP

How does a single number
make a character grow?

When xp reaches the need threshold, the game sets draft = true and switches to a separate update function that only reads card input. Choosing a card directly rewrites a stat field (speed, radius, or life). The threshold then rises, so the next level-up requires more XP.

Gem magnetism

Gems within 90 pixels of the player are pulled toward them every tick. Reaching within 18 pixels picks up the gem and increments xp.

Level-up trigger

The moment xp >= need, set draft = true. The main Update delegates to updateDraft, freezing all game simulation until a card is chosen.

Draft choice

One of three upgrade cards is selected by key press or tap. The chosen stat field is incremented, need grows by 3, and draft is cleared to resume play.

Incrementing need each level creates an accelerating curve: later levels demand more XP, making early decisions feel impactful and late-game tension feel earned.

IN THE REAL GAME

Choose → rewrite stat → resume

The draft handler reads a single key or tap event, runs a switch to apply the right stat delta, then resets the XP counter and clears the flag. No other code needs to know about levels — every system just reads the current stat values.

func (g *game) updateDraft() error {
    choice := -1
    if inpututil.IsKeyJustPressed(ebiten.Key1) { choice = 0 }
    if inpututil.IsKeyJustPressed(ebiten.Key2) { choice = 1 }
    if inpututil.IsKeyJustPressed(ebiten.Key3) { choice = 2 }
    if choice < 0 { return nil }
    switch choice {
    case 0: g.speed += .65
    case 1: g.radius += 14
    case 2: g.life++
    }
    g.level++; g.xp = 0; g.need += 3
    g.draft = false
    return nil
}

WITHOUT DRAFT

Flat games feel over too soon

Without growth, players repeat the same actions the whole run. Level-up moments create anticipation and a sense of progression that keeps players engaged.

WITH DRAFT

Each run plays differently

Choosing different card combinations each run produces distinct playstyles. The decision itself — not just the upgrade — is the fun.

YOUR FIRST TUNING

Adjust card values and XP curve

Change +.65 speed or +14 radius to feel the power difference. Raise need += 3 to make later levels a grind, or lower it for rapid-fire upgrades.
