DEEP DIVE / WAVES AND DIFFICULTY

How does one formula
create three waves of tension?

Spawn interval and enemy speed are both computed from elapsed seconds. interval = max(14, 42 - sec/2) shrinks the gap between spawns linearly. speed = 0.85 + sec × 0.018 raises enemy velocity over time. These two expressions are the entire difficulty curve — a fixed boss appears at 35 seconds as the final threat.

1

Shrinking spawn interval

Starting at 42 frames, the interval decreases by half a frame per second. A floor of 14 frames prevents infinite simultaneous spawning as time grows large.

interval := max(14, 42-sec/2)
2

Linear speed ramp

Enemy movement speed starts at 0.85 and rises by 0.018 every second. By second 35 it reaches roughly 1.48 — noticeably faster than the opening wave.

speed := .85 + float64(sec)*.018
3

Boss and HP bar

At 35 seconds a large purple mob with HP 60 is appended. The aura deals 1 HP every 14 frames. Two rectangles draw the bar: background and a proportional fill.

g.mobs = append(g.mobs, mob{..., 60, 38, true})

THE DIFFICULTY CURVE

interval := max(14, 42 - sec/2)andspeed := .85 + float64(sec) * .018

Both expressions are linear in time, so the difficulty ramp feels predictable — players can sense the pressure growing and mentally prepare for the boss. Non-linear curves (exponential or step-based) are alternatives worth experimenting with.

IN THE REAL GAME

Time → interval → speed → boss spawn

Each frame, compute elapsed seconds, derive interval and speed, then apply them. Regular mobs stop spawning at 35 seconds. The boss is slower (0.65) but has 60 HP and a proportional health bar drawn with two rects.

sec := g.frame / 60
interval := max(14, 42-sec/2)
if g.frame%interval == 0 && sec < 35 {
    g.mobs = append(g.mobs, mob{..., 1, 13, false})
}
if sec >= 35 && !g.bossSpawned {
    g.bossSpawned = true
    g.mobs = append(g.mobs, mob{240, 95, 60, 38, true})
}
speed := .85 + float64(sec)*.018
for i := range g.mobs {
    m := &g.mobs[i]
    ms := speed
    if m.boss { ms = .65 }
    m.x += (g.px - m.x) / d * ms
}

WITHOUT SCALING

Flat difficulty feels predictable

If spawn rate and speed never change, players assess the challenge within the first 30 seconds and either coast or give up — no escalating tension to keep them engaged.

WITH DIFFICULTY CURVE

Growing pressure builds to a climax

Linear scaling ensures each moment feels slightly harder than the last. The boss arrival transforms accumulated pressure into a definitive climax moment.

YOUR FIRST TUNING

Steepen the curve or raise boss HP

Change the divisor in 42 - sec/2 from 2 to 1 for a steep early ramp. Raise boss HP from 60 to 120 for an epic final confrontation.
