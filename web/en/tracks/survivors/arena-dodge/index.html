DEEP DIVE / NORMALIZED MOVEMENT

Why doesn't diagonal
movement feel faster?

Pressing right and down together gives X=1 and Y=1. Their combined length is √2 ≈ 1.41, so diagonal would be 41 % faster. Dividing both components by that length first — normalization — shrinks the arrow to exactly length 1 before multiplying by speed.

1

Input vector

Combine key states into a single direction arrow. Diagonal simply means both axes have a non-zero value at the same time.

dx += 1; dy += 1
2

Normalize

Compute the arrow's length with math.Hypot and divide both components by it. The result is a unit vector of length exactly 1.

dx /= math.Hypot(dx, dy)
3

Apply speed

Multiply the unit vector by the desired speed constant before adding to position. Direction is preserved; only the magnitude changes.

g.px += dx / l * 4

THE NORMALIZE RULE

l := math.Hypot(dx, dy)theng.px += dx / l * 4

When no key is held, dx and dy are both 0 and l would be 0 too. Always guard with if dx != 0 || dy != 0 before dividing to avoid a division-by-zero crash.

IN THE REAL GAME

Gather input → normalize → move

Both keyboard and touch are mapped to the same dx/dy pair. For touch, the arrow is simply the vector from the player to the finger. Then one normalization handles every input source identically.

if ids := ebiten.AppendTouchIDs(nil); len(ids) > 0 {
    x, y := ebiten.TouchPosition(ids[0])
    dx = float64(x) - g.px
    dy = float64(y) - g.py
}
if dx != 0 || dy != 0 {
    l := math.Hypot(dx, dy)
    g.px += dx / l * 4
    g.py += dy / l * 4
}

WITHOUT NORMALIZATION

Diagonal is unfairly fast

Without normalizing, diagonal is ≈1.41× faster. Players instinctively exploit it, breaking the intended difficulty curve.

WITH NORMALIZATION

All eight directions feel equal

Normalized movement makes every direction identical in speed, giving consistent and predictable feel — the foundation of all survivors-style games.

YOUR FIRST TUNING

Try changing the speed constant

Replace the 4 in dx / l * 4 with 6 for a nimble feel or 2 for sluggish tension. Balance it against the enemy speed constant 1.15.
