IN THE REAL GAME

Swap only after the rule passes

If the Manhattan distance is one, Go's multiple assignment exchanges both values.

distance := abs(a.x-b.x) + abs(a.y-b.y)
if distance == 1 {
    board[a.y][a.x], board[b.y][b.x] =
        board[b.y][b.x], board[a.y][a.x]
}
