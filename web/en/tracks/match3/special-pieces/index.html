DEEP DIVE / DETECT → KIND → EFFECT

Specials are two functions: shape detect, then blast set.

Special pieces are not magic. 1) Read a shape (4-run, 5-run, L/T), 2) Return the cells that kind clears. Effects differ only in how the set is built; the final clear path matches a normal match.

Keep shape detection and range math out of one giant if — that is how you grow the special roster.

IN THE REAL GAME

Split detectSpecial and effectCells

The demo picks a recipe, forges, then activates. Separating “create” from “clear” lets cascades reuse the same effectCells when another special is caught in the blast.

kind, at, ok := detectSpecial(board)
if !ok { return }
specials[at] = kind

// on activate
for p := range effectCells(kind, at, board) {
    board[p.y][p.x] = empty
}

WHY ORDER?

L before straight

An L arm is also a 3-run; detecting lines first drops the bent shape.

WHY A SET?

Overlap once

Row clear and bomb may share a cell; a map clears and scores it once.
