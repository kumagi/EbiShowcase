DEEP DIVE / PATTERN TO EFFECT

Turn a color's count and shape into a move blueprint.

A shape harder than an ordinary match of three can earn a reward. First inspect the board and name the pattern. Then build a set of affected cells. Keeping those jobs separate makes it easy to add more special pieces later.

1

Count horizontally and vertically

From one cell, count matching colors in both directions. Five can make a color wave, while four can make a row rocket.

if horizontal >= 5 { kind = ColorWave }
2

Check intersections first

If both directions reach three, the pieces form a bent L or T. Test this before a line of four or you may mistake the bend for a plain line.

Collect an effect set

IN THE REAL GAME

Separate detection from effects

detectSpecial says what was made. activate reads that kind and decides where it reaches. Avoid hiding both jobs in one long conditional.

kind, center, ok := detectSpecial(board)
if ok {
    special.kind = kind
    special.pos = center
}

for p := range effectCells(special) {
    board[p.y][p.x] = Empty
}

WHY ORDER?

Test special shapes first

An L contains a short straight line. Checking the shape with more conditions first awards the correct reward.

WHY A SET?

Make overlaps safe

If a rocket and bomb both reach one cell, a map prevents clearing or scoring that cell twice.

TRY NEXT
