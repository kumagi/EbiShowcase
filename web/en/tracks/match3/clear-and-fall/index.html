While falling, discard input — only progress

After a clear, the demo auto-plays the fall tween. Buttons do nothing until landing, then refill unlocks.

if g.falling {
    g.progress += 0.07
    if g.progress >= 1 {
        g.falling = false
        g.phase = refill
    }
    return // ignore input
}
