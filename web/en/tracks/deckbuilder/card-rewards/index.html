DEEP DIVE / DECK EDITING

A reward changes your next strategy, not just one number.

Build candidates as a list of card data, then append exactly one chosen card to deck. Rarity is data for color and appearance chance; keep it separate from raw power.

1

Build candidates

Display three cards from the reward table for this stage. Drawing and selection use the same data, so what you see is what you get.

rewards := rewardTable[stage]
2

Add only one

Append the selected card to the end of the deck. The other two disappear, making the choice meaningful.

deck = append(deck, rewards[choice])
3

Edit source data

Upgrade increases damage; removal cuts an element from the slice. The next battle reads this edited deck.

deck[i].damage += 3
