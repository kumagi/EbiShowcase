DEEP DIVE / GRAPH MAP

A map is data for places and connections, not just a picture.

Create every place as a node and store the numbers of legal destinations in next. Accept input only for current.next, so players cannot jump across a missing line.

1

Make nodes

Keep position, kind, danger, and reward in one struct. Draw reads position while event code reads kind.

type node struct { x, y float32; kind string }
2

Connect by number

If next is [3,4], either node 3 or 4 is legal. The complete web of connections is called a graph.

next []int
3

Remember visits

Mark chosen nodes in visited and color the route. A save file can store current and this history too.

visited[current] = true

IN THE REAL GAME

Test only lines leaving the current node

If a tapped circle belongs to next, move there and resolve its event once.

options := nodes[current].next
for _, id := range options {
    if tapped(nodes[id]) {
        current = id
        resolve(nodes[id].kind)
    }
}
