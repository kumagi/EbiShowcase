Build a Slay the Spire-style game

Deck-building Card Game

Build a Slay the Spire-style game where you choose cards to build your deck as you climb. Start by playing one card, then add deck circulation (draw → play → discard → reshuffle), enemy intents, status effects like poison or strength that last over turns, rewards, and a branching map.
