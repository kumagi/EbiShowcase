DEEP DIVE / MUTABLE TILE MAP

Change a cell's number and the world changes shape.

The board is a table of numbers called board[y][x]. Zero is air, one is dirt, and two is stone. Placing spends one item and writes its number; breaking writes zero and returns the material to inventory.

1

Turn screen into cell

Subtract the board's top-left from the pointer, then divide by cell width. This gives integer column x and row y.

x = (pointerX - boardX) / cell
2

Check reach and stock

Place only within three steps of the builder and when at least one selected block remains. Invalid input changes no data.

if inReach(x,y) && stock[kind] > 0
3

Pair place and break

Placing moves material from inventory to board. Breaking moves it from board to inventory. Material does not appear from nowhere.

board[y][x] = kind; stock[kind]--

IN THE REAL GAME

Pass every condition before placing

Update the board and inventory together only when the cell is empty, in reach, and stocked.

if board[y][x] == empty &&
   inReach(x, y) && stock[selected] > 0 {
    board[y][x] = selected
    stock[selected]--
}
// break: return it to inventory
stock[board[y][x]]++
board[y][x] = empty
