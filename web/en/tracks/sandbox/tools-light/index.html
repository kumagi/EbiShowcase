Check “what mines it?” and “can I see it?” before changing a tile.

The tools crafted in the previous step now store preferred material, power, and durability. Before mining, compare tool with material and inspect that tile's light. A torch is not one big bright picture: every tile calculates brightness from its distance to the source.

IN THE REAL GAME

Build maximum light from day and every torch

Brightness is an integer from 0 to 9. DAY/NIGHT and the value are printed too, so information is not carried only by color.

func (g *Game) lightAt(x, y int) int {
    light := g.daylight()
    for _, torch := range g.torches {
        distance := abs(x-torch.x) + abs(y-torch.y)
        light = max(light, 9-distance*2)
    }
    return max(0, min(9, light))
}

WHY MAX LIGHT?
