DEEP DIVE / SEEDED VALUE NOISE

Rebuild a large world from a number and coordinates.

Choosing an unrelated height in every column makes needle-like ground. This game makes the same random value from seed and coordinates, then smoothly connects neighboring values with value noise. Once height is known, depth selects grass, dirt, or stone.

1

Mix seed and coordinates

hash(seed, x, y) returns the same value whenever those three numbers match. It does not depend on generation order, so a distant location can be built directly.

n := hash(seed, x, y)
2

Smooth between samples

Use a random marker every eight columns, then change gradually between them with smooth. Heights are less likely to jump many blocks at once.

height := 3 + int(valueNoise(seed, x/8)*6)
3

Choose layers by depth

The surface cell is grass, the next three are dirt, and deeper cells are stone. Turning stone with cave noise above 0.70 back into air makes caverns.

if y > surface+3 { tile = Stone }

IN THE REAL GAME

Find height first, then stack layers

Computing one surface value per column tells us where air ends. The same surface powers material choice and the explorer's footing.

surfaceY := 3 + int(valueNoise1D(seed, float64(x)/8)*6)
for y := surfaceY; y < worldH; y++ {
    switch {
    case y == surfaceY: tiles[y][x] = grass
    case y <= surfaceY+3: tiles[y][x] = dirt
    default: tiles[y][x] = stone
    }
}
