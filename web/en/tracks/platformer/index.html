Build a Mario-style game

Side-scrolling Action

Build a side-scrolling action game like Super Mario. Start with jumping, then add moving platforms, enemies, a camera, and power-ups.

This continues the shared foundation. Each step adds one genre piece on top of the LEVEL 01 Update / Draw loop. The hero is the original Ebi Tenjiroh (海老・天次郎).
