DEEP DIVE / GROUNDED JUMP

Why can't you jump
while in the air?

The player carries a boolean flag called grounded. It is set to true only when the player lands on solid ground and reset to false every single frame before collisions are checked. Jump input is ignored unless grounded is true, so double-jumping is impossible without extra code. Gravity is simulated by adding a small constant to vy each frame — just one line.

1

Gravity

Add +0.65 to vy every frame. Without any input the player accelerates downward naturally. A cap of 14 prevents infinite speed.

vy = min(vy+.65, 14)
2

Ground detection

After moving vertically, check each block for overlap. If falling (vy > 0) and overlapping, snap the player to the block top and set grounded = true.

grounded = true
3

Axis separation

Move X, resolve X collisions, then move Y, resolve Y collisions. Doing both at once causes corner-catching bugs where the player gets stuck on platform edges.

p.x += vx // horizontal first

THE ONE-LINE RULE

if jump && grounded { vy = -12 } every frame grounded = false → collision check sets it back

Reset grounded to false at the top of each frame, then let the collision loop set it back to true when a landing is detected. Being on the ground must be proved anew every frame.

IN THE REAL GAME

Vertical collision
decides grounded

When falling (vy >= 0) and overlapping a block, snap to the top surface and set grounded = true. When rising (vy < 0) and hitting a ceiling, snap to the bottom surface. Both paths clear vy.

g.p.y += g.vy
g.grounded = false
for _, b := range g.blocks {
    if overlap(g.p, b) {
        if g.vy > 0 {
            g.p.y = b.y - g.p.h // land on top
            g.grounded = true
        } else {
            g.p.y = b.y + b.h   // bump ceiling
        }
        g.vy = 0
    }
}

WITHOUT GROUNDED FLAG

You could jump forever in mid-air

Without the flag, any jump press would always apply upward velocity. Game balance breaks and any height-based puzzle becomes trivial.

WITH SEPARATE AXES

Corner collisions resolve cleanly

Splitting the axes avoids the classic corner-catch bug where the player gets pinned when sliding along a platform edge.

YOUR FIRST TUNING

Tune jump power and gravity

Increase g.vy = -12 for a floatier jump, or raise the +.65 gravity constant for a snappier, heavier feel. The ratio between the two defines the jump arc.
