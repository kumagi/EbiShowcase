HOW IT WORKS

A game combines small systems

Movement, jumping, enemies, coins, and transformations are separate small rules. Call them in order from Update, then increase the stage number and load new data at the goal.

updateMovement()
updateCollectibles()
updateEnemies()
if reachedGoal {
    stage++
    loadStage(stage)
}
