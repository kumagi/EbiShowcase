Project 2D distances into a 3D-looking view

Ray, distance, and projection are pure functions separated from Ebitengine, so go test can check them without opening a window.

for column := 0; column < width; column += 2 {
    rayAngle := left + float64(column)/width*fov
    hit := Cast(world, player, rayAngle)
    drawWallStrip(column, hit.Distance)
}
