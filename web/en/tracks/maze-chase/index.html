Build a Pac-Man-style chase

Maze Chase

Build a Pac-Man-style game where you eat pellets in a maze while running from enemies. Start with tile movement, then add remembering your next turn early (buffered input), intersections, enemy modes, path choice, and power states.

This continues the shared foundation. Each step adds one genre piece on top of the LEVEL 01 Update / Draw loop. When a hero appears, it is the original Ebi Tenjiroh (海老・天次郎).
