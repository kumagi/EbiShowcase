Give each enemy a personality

Junction AI

At each junction, an enemy finds legal directions, removes the way back, and picks the step closest to its target. Change only the target data and its chasing style changes.

★★★★☆PLAYABLE

PLAY FIRST

Dodge two predictions and collect eight chips

Move with arrows/WASD, clicks, or touch. The red HUNTER targets your current tile; the cyan AMBUSHER targets three tiles ahead. Colored rings show targets and yellow lines show chosen directions.

DEEP DIVE

Split chasing into candidates, evaluation, and personality

Do not move an enemy straight toward the player. First collect choices that obey the maze. Measure the Manhattan distance from each next tile to the target, then choose the smallest score.

01

List legal directions

Inspect the four neighboring tiles and keep only directions that are not walls.

02

Remove reverse

When several choices exist, remove the way it came from. Keep reversal at a dead end so it never gets stuck.

03

Score a personality target

HUNTER targets now; AMBUSHER targets three tiles ahead. Both choose the candidate with minimum distance.

GO + EBITENGINE

Pass personality targets into one chooser

An ahead value of 0 means the current tile; 3 means three tiles along the player's direction. One algorithm supports different enemies through data.

target := player
if enemy.personality.ahead > 0 {
    target = player.add(playerDirection.mul(3))
}

legal := legalDirections(enemy.position)
if len(legal) > 1 {
    legal = removeReverse(legal, enemy.direction)
}
best := legal[0]
for _, dir := range legal[1:] {
    if distance(enemy.position.add(dir), target) <
       distance(enemy.position.add(best), target) {
        best = dir
    }
}

Why forbid reversal?

Picking only the nearest move can make an enemy bounce between two tiles. Moving onward at junctions makes pursuit look intentional.

How is it strategy?

Turn away to build distance from HUNTER. Let AMBUSHER predict a straight run, then enter another corridor early. Reading personality becomes play.
