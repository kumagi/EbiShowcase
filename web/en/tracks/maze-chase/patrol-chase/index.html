DEEP DIVE / ENEMY MODE MACHINE

Change an enemy's destination to change its personality.

The enemy owns mode and target. PATROL targets the next tile in a route, CHASE targets the current player, and SEARCH targets the last seen tile. Distance and line of sight switch modes.

1

PATROL follows a stored route

patrolRoute stores corner tiles in order. On arrival, increment index and wrap to zero at the end.

target = patrolRoute[patrolIndex]
2

Straight sight starts CHASE

The enemy sees when x or y matches, no # lies between, and distance is at most seven. Then player becomes the target.

if lineOfSight && distance <= 7 { mode = chase }
3

Losing sight starts SEARCH

Update lastSeen while visible. After a corner breaks sight, head there, then return to patrol on arrival or timeout.

target = lastSeen

IN THE REAL GAME

Separate transition from movement

updateMode first selects state and target. moveEnemy then takes one tile toward that target. The label reads the same mode value.

visible := hasLineOfSight(enemy, player) &&
           manhattan(enemy, player) <= 7
if visible {
    mode = chase
    lastSeen = player
} else if mode == chase {
    mode = search
}

switch mode {
case patrol: target = patrolRoute[index]
case chase:  target = player
case search: target = lastSeen
}
