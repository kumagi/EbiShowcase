DEEP DIVE / DROP TIMER

Do not move every frame. Drop only when the clock fills.

Ebitengine calls Update about 60 times each second. A block that moved every Update would reach the floor instantly. Instead, timer counts how many updates have passed, and the cell moves down once when the count reaches its limit.

1

Count waiting updates

Add one to timer on every Update. The yellow bar below the board turns that invisible count into a clock the player can read.

timer++
2

Move by grid cells

Only when timer reaches 38, add one to row y and reset timer. Position is an integer row and column, not a loose decimal coordinate.

if timer >= 38 { y++; timer = 0 }
3

Lock when blocked

If the next cell is the floor or an occupied board cell, write the active cell into board[y][x], then spawn a new one at the top.

board[y][x] = true; spawn()
