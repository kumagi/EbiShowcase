Build a Tetris-style game

Falling-block Puzzle

Build a Tetris-style puzzle where you stack falling pieces to clear full rows. Grow one falling block into rotation, locking, line clears, hold, and modern scoring.

This continues the shared foundation. Each step adds one genre piece on top of the LEVEL 01 Update / Draw loop. When a hero appears, it is the original Ebi Tenjiroh (海老・天次郎).
