DEEP DIVE / FAIR QUEUE AND PREDICTION

Turn invisible luck into visible data.

With independent random picks, one needed shape might disappear for dozens of turns. A seven-shape bag shuffles one of each and prevents huge droughts. HOLD and the ghost then let players make a plan instead of only reacting to luck.

IN THE REAL GAME

The ghost never changes the board

It calls the same canPlace as the real piece, but changes only a local variable. Drawing a prediction does not advance game state.

func (g *Game) ghostY() int {
    y := g.y
    for g.canPlace(g.kind, g.rotation, g.x, y+1) {
        y++
    }
    return y
}
