DEEP DIVE / ROTATE THEN TEST

Rotate first, then ask, “Does it fit here?”

Treat a piece's center as (0, 0). A cell at (x, y) moves to (-y, x) after a clockwise quarter turn. The rotated piece may poke through a wall, so test a list of small kick offsets.

1

Rotate all four cells

T and L pieces are four offsets from a center. Apply the same transform to every offset.

rotated = point{-p.y, p.x}

Test candidates in order

Try no shift, left one, right one, left two, right two, then up one. Accept the first position without a collision.

kicks := []point{{0,0}, {-1,0}, {1,0}}

Keep the old pose if blocked

IN THE REAL GAME

Test rotation and offset together

Build the four rotated cells at each candidate position. Commit only the first pair for which collides returns false.

nextRotation := (rotation + 1) % 4
for _, kick := range kickTests {
    candidate := pos.add(kick)
    if !collides(candidate, nextRotation) {
        pos = candidate
        rotation = nextRotation
        break
    }
}

WHY A CENTER?

One formula fits every shape

Offsets from a center let T and L pieces share the same rotation calculation.

WHY AN ORDER?

Avoid surprise teleports

Trying zero first means the smallest necessary kick wins instead of moving the piece too far.
