Build a Bomberman-style blast game

Bomb Maze Action

Build a Bomberman-style game where you drop bombs in a maze to blast walls and enemies. Start with timed bombs, then add a cross-shaped blast that grows up, down, left, and right until it hits a wall, breakable walls, chain reactions, items, and enemy AI.

This continues the shared foundation. Each step adds one genre piece on top of the LEVEL 01 state boundary. When a hero appears, it is the original Ebi Tenjiroh (海老・天次郎).
