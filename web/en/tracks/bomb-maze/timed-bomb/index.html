Place a Timed Bomb

GO + EBITENGINE

Update timer and state

Switch state only on the frame the remaining time reaches zero.

for i := range bombs {
    bombs[i].timer--
    if !bombs[i].blasting && bombs[i].timer <= 0 {
        bombs[i].blasting = true
        bombs[i].timer = blastLife
    }
}

Why use a slice?

Why restrict placement?
