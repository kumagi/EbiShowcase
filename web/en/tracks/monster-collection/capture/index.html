DEEP DIVE / VISIBLE PROBABILITY

Let preparation improve chance instead of relying on luck alone.

Capture chance is always visible. Start at 15%, add up to 55% from missing HP, 20% for SLEEP, and 15% for a REEF ORB. Finally compare one 0–99 roll with that chance.

IN THE REAL GAME

Keep chance calculation separate from saving

captureChance only returns a number. tryCapture draws the roll and updates storage only on success.

func captureChance() int {
    hpBonus := (maxHP-wildHP) * 55 / maxHP
    statusBonus := 0
    if sleeping { statusBonus = 20 }
    return min(95, 15+hpBonus+statusBonus+itemBonus)
}

if rng.Intn(100) < captureChance() {
    storage = append(storage, wildName)
}
