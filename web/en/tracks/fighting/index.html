Build a fighter from collision boxes

2D Fighting Game

Build a one-on-one arcade-style fighting game where you and an opponent trade moves. Start with pushboxes (boxes that keep characters from overlapping), hurtboxes (boxes that can get hit), and hitboxes, then add move frames, hitstop (freezing time for an instant on impact), and combos.

This continues the shared foundation. Each step adds one genre piece on top of the LEVEL 01 state boundary. The hero is the original Ebi Tenjiroh (海老・天次郎).
