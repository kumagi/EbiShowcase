HOW IT WORKS

Freeze time and disabled time are different

During hit stop, end the whole Update immediately. Then block only the opponent during hit stun and add knockback velocity to move them backward.

if hitStop > 0 { hitStop--; return nil }
if hitStun > 0 {
    enemy.X += knockbackVelocity
    hitStun--
}

GAME + LESSON

01

Freeze

02

Disable

Ignore opponent input during hit stun.

03

Push

Move backward with a shrinking velocity.
