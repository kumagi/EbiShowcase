DEEP DIVE / HIT REACTION

Why does a hit
feel so heavy?

The answer is hitstop. Freezing the world for a few ticks makes the brain register “that connected.” Then hitstun locks control and knockback slides the foe away.

1

Hitstop

Freeze motion for a few ticks on hit — but read input first into bufAttack so presses during the freeze still start the next attack.

if start { bufAttack = true }
2

Hitstun

While stunned, disable normal AI and only allow the reaction motion.

if stun > 0 { stun--; p2 += v2 }
3

Knockback

Give push velocity v2, then damp it each tick so the slide eases out.

v2 = 7; v2 *= .86

THE REACTION CHAIN

hitstop = 8 then stun = 25; v2 = 7

Order matters. Freeze first, then set stun and knockback. Longer freeze feels heavier; shorter feels snappier.

IN THE REAL GAME

Set all three
on the hit frame

On the active attack frame, set hitstop, stun, and knockback. Still sample input during hitstop into bufAttack, then return — only simulation freezes.

if start { g.bufAttack = true }
if g.hitstop > 0 {
    g.hitstop--
    return nil // freeze motion; input already saved
}
if g.bufAttack && g.attack == 0 {
    g.attack = 20
    g.bufAttack = false
}
if g.attack == 12 && g.p2-g.p1 < 115 {
    g.hitstop = 8
    g.stun = 25
    g.v2 = 7
}
