DEEP DIVE / HIT REACTION

Why does a hit
feel so heavy?

The answer is hitstop. Freezing the world for a few frames makes the brain register “that connected.” Then hitstun locks control and knockback slides the foe away.

1

Hitstop

On hit, early-return from Update for a few frames. A background tint makes the freeze visible.

if hitstop > 0 { hitstop--; return nil }
2

Hitstun

While stunned, disable normal AI and only allow the reaction motion.

if stun > 0 { stun--; p2 += v2 }
3

Knockback

Give push velocity v2, then damp it each frame so the slide eases out.

v2 = 7; v2 *= .86

THE REACTION CHAIN

hitstop = 8 then stun = 25; v2 = 7

Order matters. Freeze first, then set stun and knockback. Longer freeze feels heavier; shorter feels snappier.

IN THE REAL GAME

Set all three
on the hit frame

On the active attack frame, check range. On a hit, set stop, stun, and velocity together.

if g.attack == 12 && g.p2-g.p1 < 115 {
    g.hitstop = 8
    g.stun = 25
    g.v2 = 7
    g.hits++
}
