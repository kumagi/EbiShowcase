THE MATCH RULE

round ends → award win then wins >= 2 → match over

Best-of-three is kinder for learning — you can recover after a lost round.

IN THE REAL GAME

Tally wins from round results

Compare HP, award the round, end at two wins or start the next round.

if g.p.hp > g.ai.hp {
    g.pWins++
} else {
    g.aiWins++
}
if g.pWins >= 2 || g.aiWins >= 2 {
    g.matchOver = true
} else {
    g.round++
    g.resetRound()
}
