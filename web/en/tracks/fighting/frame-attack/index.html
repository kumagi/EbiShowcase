HOW IT WORKS

Divide move time with frame numbers

Frames 1–8 prepare, 9–12 own an attack box, and 13–30 recover. Pressing attack does not mean hitting immediately.

switch {
case frame <= 8:  phase = Startup
case frame <= 12: phase = Active
case frame <= 30: phase = Recovery
}

GAME + LESSON

Give time three jobs

01

Startup

Preparation before an attack can hit.

02

Active

A short time when the red attack box exists.

03

Recovery

Return time before another action is possible.
