HOW IT WORKS

Check overlapping rules in one fixed order

Guard beats strike, throw beats guard, and strike beats throw preparation. A single priority table keeps simultaneous conditions consistent.

func wins(a, b Action) bool {
    return a==Guard && b==Strike ||
           a==Throw && b==Guard ||
           a==Strike && b==Throw
}
