IN THE REAL GAME

Branch work by scene

Update battle while fighting; otherwise move on the field and test encounters.

if g.scene == battle {
    return g.updateBattle()
}
g.walk()
if g.shouldEncounter() {
    g.enemy = g.rollEnemy(g.region)
    g.scene = battle
}
