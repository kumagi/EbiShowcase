Build a Dragon Quest-style game

Command RPG

Build a Dragon Quest-style RPG where you pick a command each turn. Start with town exploration and dialogue, then add turn-based combat (taking turns to pick Attack, Defend, or Item), growth, equipment, and a world map.

This continues the shared foundation. Each step adds one genre piece on top of the LEVEL 01 Update / Draw loop. The hero is the original Ebi Tenjiroh (海老・天次郎).
