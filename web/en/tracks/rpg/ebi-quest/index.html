HOW IT WORKS

The quest number is a story bookmark

Quest 0 finds a companion, 1 finds the crystal, 2 fights the boss, and 3 returns home. Save it with position and HP whenever it changes so reopening continues from that bookmark.

type SaveData struct { X, Y, HP, Quest int }
quest++
save(SaveData{player.X, player.Y, player.HP, quest})
