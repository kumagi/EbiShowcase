THE SAVE RULE

save(quest, x, y, hp) on resume load() → 同じ世界

Save the minimum. You rarely need every enemy position — a quest id rebuilds the story.

IN THE REAL GAME

Pair save and load

load on boot, save after events. R can wipe keys for a fresh start.

func (g *game) save() {
    s := js.Global().Get("localStorage")
    s.Call("setItem", saveKey+"quest", strconv.Itoa(g.quest))
    s.Call("setItem", saveKey+"x", strconv.Itoa(g.x))
    s.Call("setItem", saveKey+"y", strconv.Itoa(g.y))
    s.Call("setItem", saveKey+"hp", strconv.Itoa(g.hp))
}
