DEEP DIVE / DAMAGE FORMULA

Damage is decided
by subtraction

The core formula is damage = attack − defense. Layering temporary effects on top is straightforward: a countdown integer tracks how many turns a buff or debuff lasts. Decrement it each turn; when it reaches zero the effect disappears automatically. No special timer or separate cleanup code is needed.

1

The damage formula

Subtract defense from attack, add a random spread, then clamp to a minimum of 1. max(1, ...) prevents high-defense enemies from taking zero damage.

d := max(1, atk - def + rand)
2

Buff as a countdown

Power Up sets buff = 3. On attack, if buff > 0 add bonus damage. Decrement buff at the end of each turn. It vanishes after three turns with no extra cleanup code.

if g.buff > 0 { atk += 8 }
3

Poison as a countdown

Every third enemy turn inflicts poison = 4. While poison > 0 the player takes 3 damage per turn and poison decrements. Antidote sets it to zero immediately.

if g.poison > 0 { g.hp -= 3; g.poison-- }

THE DAMAGE FORMULA

atk := g.attack // base 15 apply power-up bonus if g.buff > 0 { atk += 8 } subtract defense, add spread d := max(1, atk - 6 + g.rng.Intn(5))

The enemy's defense is the fixed value 6. Base attack 15 − defense 6 = 9 minimum damage. With Power Up that becomes 17 minimum. Changing those two numbers reshapes the entire difficulty curve.

IN THE REAL GAME

Attack, buff, and poison in one Update pass

After the player chooses a command, the code applies the player's action, runs the enemy counter-attack, then updates both countdown integers. The order matters: buff is read before attack so the bonus applies in the same turn it was started.

// player chose Attack
atk := g.attack
if g.buff > 0 { atk += 8 }
d := max(1, atk-6+g.rng.Intn(5))
g.enemyHP -= d

// enemy counter-attack and status updates
enemyAttack := 13 + g.rng.Intn(5)
g.hp -= max(1, enemyAttack-g.defense)

if g.turn%3 == 0 { g.poison = 4 }
if g.poison > 0 { g.hp -= 3; g.poison-- }
if g.buff > 0 { g.buff-- }

WITHOUT FORMULA

Fixed damage means no strategy

If every hit deals the same amount, equipment and buffs have no purpose. The attack-minus-defense formula is what makes equipping a better sword feel meaningful.

WITH COUNTERS

Countdowns make effects expire naturally

Permanent buffs break game balance. A countdown integer gives every effect a finite lifespan and communicates it clearly on-screen: "POWER UP 2" means two turns remain.

YOUR FIRST TUNING

Change buff duration and strength

Set buff = 5 and atk += 12 to make Power Up last longer and hit harder. Or set enemy defense to 12 and watch the damage floor drop — then think about how to compensate.
