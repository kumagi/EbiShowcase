How does a picture

In Ebitengine you give a drawing a transform, then call screen.DrawImage. The most basic transform is Translate. You can stamp the same image many times just by changing the Translate numbers.

IN THE REAL GAME

One image, any place

Make an options box, set the position with Translate, then draw.

op := &ebiten.DrawImageOptions{}
op.GeoM.Translate(x, y) // 左上を (x, y) へ
screen.DrawImage(tenjiroh, op)

WHY OPTIONS?

Carry the how

Position, color, and blend all travel in one box.

WHY TRANSLATE?

Position is a transform

The image never changes—only where you place it.
