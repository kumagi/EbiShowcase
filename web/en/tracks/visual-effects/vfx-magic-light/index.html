IN THE SHOWCASE

Pillar → halo → benediction

Direction, symmetry, and slow particles give the effect a ceremonial order.

DrawLightPillar(target, whiteGold)
DrawHalo(target, rings: 3)
for i := 0; i < rays; i++ {
  ang := -math.Pi/2 + float64(i)*2*math.Pi/float64(rays)
  DrawSymmetricRay(target, ang)
}
for _, mote := range blessingMotes { mote.y += fallSlowly }

WHY FROM ABOVE?

Give the light a source

Descending toward the target feels bestowed; exploding from the center feels like another attack.

WHY SYMMETRY?

Order becomes sacred

Still rings, a cross, and paired rays read as an emblem or ritual rather than random glare.
