VFX

Sparks, light, walk cycles—a look toolbox

Visual Effects Lab

This lab practices “how it looks,” not game rules. Touch one drawing tool at a time—stamp, rotate, tint, glow, walk. Later chapters add sparkle to the core LEVEL mini games you already played. You don’t need every step—pick a title that looks fun.

This continues the core lessons (LEVEL 01–12). On top of the Update / Draw loop, add one presentation tool at a time. The hero is original Ebi Tenjiroh (海老・天次郎 / ebi-tenjiroh).

FIRST-TIME READER

Read the game as small parts

Update changes Ebi Tenjiroh's game state; Draw shows that state as the next frame. In func (g *Game), (g *Game) is the receiver—the name tag for this Game.

Ebi Tenjiroh's view
Update changes Tenjiroh's numbers; Draw paints the next frame.

(g *Game) is the receiver: a name tag that lets the function use this Game's state.

LEARNING PATH

Basics, then Advanced.

Basics: feel place/spin/tint/glow with on-page sliders. Advanced: practice keeping score/hits in one box and fireworks in another. Starting at Stamp is enough.
