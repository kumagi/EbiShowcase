VFX

Make on-screen magic with Ebitengine

Visual Effects Lab

Basics teach the drawing grammar; Advanced remakes each LEVEL 01–12 mini game with flashy FX. The goal is noticing for yourself that effects belong in a separate system from gameplay entities.

This continues the core lessons (LEVEL 01–12). On top of the Update / Draw loop, add one presentation tool at a time. The hero is original Ebi Tenjiroh (海老・天次郎).

LEARNING PATH

Basics, then Advanced.

Learn the tools with LIVE GO, then dress up all twelve core games while discovering the play-vs-fx split.
