Speed, action gauges, and waiting

Here is the central rule used by the playable game.

actor.gauge += actor.speed
if actor.gauge >= maxGauge {
    actor.ready = true
}
// command resolved
actor.gauge = 0

Why this structure?

Actors do not wait for a shared turn. Faster actors fill sooner and may act more often. The gauge stores waiting time as a number.
