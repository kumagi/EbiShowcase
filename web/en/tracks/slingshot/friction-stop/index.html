DEEP DIVE / DAMP AND SNAP

Reduce motion gently, then end at a firm zero.

With friction 0.978, speed becomes 97.8% of the previous frame. Multiplying forever only approaches zero, so when speed falls below a stop threshold, assign zero and finish the turn.

1

Move by current velocity

Change position by velocity first. If the puck reaches a wall, reverse that direction before continuing.

pos.x += velocity.x
2

Multiply by friction

0.990 slides far while 0.960 slows quickly. Multiply x and y equally to keep the direction while shrinking motion.

velocity.x *= friction

IN THE REAL GAME

The order is move, damp, then test

Because the threshold check follows friction, a velocity that becomes tiny this frame snaps immediately.

pos.x += velocity.x
pos.y += velocity.y
velocity.x *= friction
velocity.y *= friction

if math.Hypot(velocity.x, velocity.y) < stopThreshold {
    velocity = vec{}
    finishShot()
}

WHY MULTIPLY?

Fast motion loses more

A percentage removes a lot at first and a little near the end, making a natural-looking slowdown.

WHY A THRESHOLD?

Prevent endless turns

Fractional velocity never quite reaches zero, so the game needs a number meaning “close enough to stopped.”
