DEEP DIVE / WALL NORMAL

Reverse only the speed pointing into the wall.

Velocity has a horizontal vx and vertical vy. A vertical wall changes only horizontal motion; a floor or ceiling changes only vertical motion. The arrow pointing straight out of a wall is its normal, and reflection flips the velocity along that normal.

1

Include the radius

Waiting for the center to touch would sink half the circle into the wall. Contact begins when the left edge, center minus radius, crosses the wall.

if x-radius < leftWall
2

Reverse one component

A side wall changes only the sign of vx. Keeping vy makes the outgoing angle match the incoming angle.

vx = -vx
3

Lose a little energy

Multiply the reflected speed by 0.92. Restitution 1 keeps all speed; a value near 0 gives most of it to the wall.

vx = -vx * restitution

IN THE REAL GAME

Clamp inside before reflecting

A circle left inside the wall collides again next frame and flips repeatedly. Move it to the boundary first, then guarantee an inward velocity.

if x-radius < leftWall {
    x = leftWall + radius
    vx = math.Abs(vx) * restitution
    bounces++
}
if y-radius < topWall {
    y = topWall + radius
    vy = math.Abs(vy) * restitution
    bounces++
}

WHY CLAMP?

Leave nothing in the wall

Returning to the boundary lets one collision count exactly once.

WHY TWO PARTS?

Preserve the angle

Motion parallel to the wall stays unchanged, making a mirror-like path.

TUNE IT
