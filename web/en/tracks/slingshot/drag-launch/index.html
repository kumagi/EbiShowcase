DEEP DIVE / OPPOSITE VECTOR

Flip the pull arrow with a minus sign.

Store the difference from the pearl's center to the pointer as a drag vector. On release, put a minus sign on both X and Y. That makes launch velocity point exactly opposite the pull.

1

Subtract two points

Subtract the pearl position from the pointer. Pulling right makes dragX positive; pulling down makes dragY positive.

dragX = pointerX - ballX
2

Limit long pulls

If length exceeds 160, shrink the vector without changing its direction. Pulling beyond the screen cannot create wild speed.

drag *= 160 / length
3

Launch oppositely

Only on release, negate dragX and dragY to make velocity. A pearl pulled downward launches upward.

velocity = -drag * power

IN THE REAL GAME

Choose velocity once on release

Update the aim while dragging, then convert it to velocity in the one frame where the pointer is released.

dragX := pointerX - ballX
dragY := pointerY - ballY
if released {
    vx = -dragX * 0.092
    vy = -dragY * 0.092
    moving = true
}

WHY OPPOSITE?

The elastic return direction

The farther you pull back, the stronger the pearl travels toward its starting side.

WHY ON RELEASE?

Separate aiming from flight

The pearl stays still while aiming and commits to the chosen direction only when released.
