HOW IT WORKS

Advance the same small time each update

Ebitengine normally calls Update sixty times per second. Add the same gravity to vertical speed, then speed to position. At the floor, restore position and reverse a smaller velocity.

velocityY += gravity
positionY += velocityY
if positionY+radius > floor {
    positionY = floor-radius
    velocityY *= -bounce
}
