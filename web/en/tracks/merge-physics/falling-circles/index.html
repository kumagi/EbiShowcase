THE TWO-LINE RULE

vy = vy + gravity then y = y + vy

A constant timestep keeps motion stable. Relying on variable frames alone makes physics wobble.

IN THE REAL GAME

Gravity → move → floor

Update unsettled circles; bounce on the floor and mark settled when slow enough.

b.vy += .55
b.y += b.vy
if b.y+b.r > 660 {
    b.y = 660 - b.r
    b.vy = -b.vy * .45
    if math.Abs(b.vy) < .8 {
        b.settled = true
    }
}
