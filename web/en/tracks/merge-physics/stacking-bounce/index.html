DEEP DIVE / IMPULSE

Do heavy and light circles bounce the same way?

Large circles get more mass. Divide collision impulse according to mass so lighter circles move farther.

1

Correction

Share overlap according to inverse mass.

overlap := radii-distance
2

Mass

Use radius squared as circle weight.

mass = radius * radius
3

Impulse

Reflect only velocity along the contact direction.

velocity += impulse / mass

Reverse velocity at a surface

IN THE REAL GAME

Correct position before velocity

Remove overlap first, then apply bounce only while circles approach.

correctPenetration(a, b) / relative := dot(b.velocity-a.velocity, normal) / if relative < 0 { applyImpulse(a, b, normal) }

WHY CORRECT FIRST?

Prevent sinking

Velocity alone leaves circles overlapping.

WHY MASS?

Size changes motion

Light circles move more than heavy circles.
