12 playable mini games in your browser

Play more.

This site is a practice ground for making games yourself. Play tiny games in the browser and learn Go plus the game library Ebitengine, one idea at a time. The hero is original character Ebi Tenjiroh (海老・天次郎 / ebi-tenjiroh)—a shrimp-themed adventurer. Start at LEVEL 01 with the simple loop: advance numbers, then draw the picture.

Read the game as small parts

Update changes Ebi Tenjiroh's game state; Draw shows that state as the next frame. In func (g *Game), (g *Game) is the receiver—the name tag for this Game.

Update changes Tenjiroh's numbers; Draw paints the next frame.
