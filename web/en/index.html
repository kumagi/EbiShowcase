BASIC / CORE LESSONS

Start with one idea.
Build toward a full game.

Each level adds only one or two ideas. What you learn in one game becomes a building block for the next.

01 / 60 FRAMES

A game is a flipbook of about 60 pictures each second.

For each picture, Update advances numbers such as position and score. Then Draw paints the picture described by those numbers—just like TV and animation swap many images in a short time.

02 / COLLISION

A collision test asks, “Do these two shapes overlap?”

A circle and a finger, a basket and a star, or a hero and a wall: compare their shapes as numbers, then run a score or stop rule when they overlap.
