This site starts with a tiny game you can play first, then take apart one idea at a time. Nothing needs installing just to play. Afterward, you can read the Go code and learn how the Ebitengine game library turns rules into a game.

BASIC / CORE LESSONS

Play first.
Then unpack the “why.”

LEVEL 01 only asks you to press a target. After playing, learn one or two ideas behind that movement. Each idea becomes a building block in the next game.

01 / PLAY

It is fine to play before reading anything.

Open LEVEL 01 and press the glowing target. You do not need to install anything or type code yet.

02 / NOTICE

Split what you saw into one rule at a time.

Name simple events such as “a hit adds a point” and “the target moves somewhere else.”

03 / MAKE

Then connect the explanation to Go code.

Read the part that changes the game separately from the part that shows it. Set up your computer when you are ready to write your own version.
