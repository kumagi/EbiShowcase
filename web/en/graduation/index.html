MAKE / GRADUATION PROJECTS

Turn learned parts into your game.

Build three small finished games from red tests. Graduation means you can change a rule and prove it, not only play a demo.

BEFORE YOU START

What you need

Use Go, a browser, and a text editor. Do not clone this repository. Each brief links to three starter files; save them from their Raw browser view into an empty folder.

mkdir my-ebi-game
cd my-ebi-game
go mod init example.com/my-ebi-game
go get github.com/hajimehoshi/ebiten/v2
go test

The first red go test is success: read the test name, complete its TODO, turn it green, then play with go run ..
