GRADUATION 03

Add data to add puzzles.

This three-stage puzzle adds names and targets to an array instead of copying the rule.

0. DOWNLOAD

Start in an empty Go folder

Save the three starter files from their Raw browser view. Run go mod init example.com/three-puzzles, go get github.com/hajimehoshi/ebiten/v2, then go test. No clone is required.

1. START

Select the first data record

TestFreshProgressStartsAtFirstDataStage maps to TODO 1. Make empty Progress point Stage 0 at Stages[0].

2. ADVANCE

Read the target from data

TestPuzzleTargetAdvancesOneStage and TestLastDataStageClearsInsteadOfOverflowing map to TODO 3. Count Presses, compare the current Stages[p.Stage].Target, advance one stage, and Clear instead of overflowing at the last record.

3. RESET

Return progress to the start

TestRestartMakesFreshProgress maps to TODO 2. Use NewProgress() so old stage, presses, and Clear do not remain.
