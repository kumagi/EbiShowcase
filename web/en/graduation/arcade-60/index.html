GRADUATION 01

Turn red tests into a 60-second game.

Start in an empty Go workspace. Download only three starter files in the browser, then turn each failing test green in TODO order.

0. DOWNLOAD

Start without cloning

In an empty folder, run these commands, then save the three starter files from their Raw browser view into that folder.

mkdir arcade-60 && cd arcade-60; go mod init example.com/arcade-60; go get github.com/hajimehoshi/ebiten/v2; go test

The first failures are correct. Do not open reference yet.

1. SCORE

Make the first red test green

When TestActionAddsOneStar fails, work on TODO 4 in rules.go. Add 10 to Score only for Input{Action: true}. main.go reads a key or click; Round.Step owns the score rule.

go test -run TestActionAddsOneStar

2. TIMER

End after 60 seconds

TestTimeLimitEndsRound maps to TODO 5. Advance Frames once per Step, then set Over when it reaches RoundFrames. Draw only displays that result.

go test -run TestTimeLimitEndsRound

3. RESTART

Make one more try

TestRestartMakesFreshRound maps to TODO 3. The R key returns to NewRound(). The test protects against carrying old Score or Frames into the new game.

go test; go run .

ADD ONE RULE

Choose a combo, a missed-action penalty, or a last-ten-seconds effect, and write its test first.

KEEP IT PURE

Rules live in rules.go; input and drawing live in main.go. You can trust a rule without reopening a window.

COMPARE LAST

Only after every test is green, open reference to observe structural differences—not to copy it.
