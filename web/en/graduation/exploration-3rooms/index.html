Carry a key through three rooms.

Put room number, key, and escape in one State, then turn each red test green in order.

From an empty Go folder

Save the three starter files from their Raw browser view. Run go mod init example.com/three-rooms, go get github.com/hajimehoshi/ebiten/v2, then go test. No clone is needed.

Collect a key only in room 1

2. ROOMS

TestPathMovesThroughExactlyThreeRooms maps to TODO 4. Move 1→2→3 for NextRoom, and stop there.

3. EXIT + RESTART

Make a key-gated exit

TestExitNeedsKeyInThirdRoom maps to TODO 5: only Room 3 plus HasKey sets Escaped. Then satisfy TestRestartMakesNewAdventure with TODO 2 and return to NewAdventure.
