FIRST 30 MINUTES

準備が終わったPCで、
30分の最初の制作。

Goのインストール時間は30分に含めません。環境づくりが終わった状態から、動くコードを一つ変え、テストを一つ通すところまで進みます。途中で時間が来たら、そのカードを今日のゴールにして大丈夫です。

始める前

準備を確かめる

まだならセットアップを先に終えます。準備済みなら go version を確認してタイマーを開始します。

00–10

動く土台を保存する

Build STEP 01 の main.go を自分のフォルダに保存し、go run . で空の窓を開きます。

10–22

得点ルールを変える

Build STEP 03 の完全な main.go で今のファイルを置き換え、その中の addScore を2点ルールへ変えます。画面で SCORE 02 を確認します。

22–30

テストで変更を守る

STEP 03のテストを期待値4へ直して go test ./...。緑になったら、次に足したいルールをコメントで1行残します。
