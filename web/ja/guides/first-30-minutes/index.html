FIRST 30 MINUTES

30分で、最初の自分のゲームへ。

読むだけで終えず、手元の Go ファイルとテストに差分を残します。

00–05

空の窓を出す

セットアップでローカル Go 環境を用意し、空の Ebitengine 窓を動かす。

05–15

1つ書く

Build Track のSTEPを進め、手元の main.go に差分を残す。

15–25

規則をテストする

テスト教材で最小の純粋関数と go test を作る。

25–30

最初の赤を緑にする

60秒アーケードをブラウザから保存し、最初の赤テストをTODOで緑にする。cloneは不要。
