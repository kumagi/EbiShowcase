01

ローカル

`go test ./...`。PCとスマホ幅で操作し、キーボード・マウス・タッチを1回ずつ試します。

02

WASM

`GOOS=js GOARCH=wasm go build`。HTMLは置き場、ゲーム本体はGo/EbitengineのWASMです。

03

Pages

ActionsのartifactとSettings → Pagesを確認。公開URLの `/ja/` と `/en/` を開きます。

04

ライセンス

自分のコードはApache-2.0。外部フォント・依存・素材は `THIRD_PARTY_NOTICES.md` に元の権利を書きます。

05

リリース
