SYSTEM GUIDE / PERFORMANCE

速くする前に、
何を測るか決めよう。

ゲームの「重い」は一つではありません。tickごとの確保、見えないものの描画、全員どうしの当たり判定を分けて測ります。

3つの小さな道具

01

再利用する slice

画面内のものを `dst[:0]` に詰め直します。今回のCullベンチは 0 allocs/op でした。

02

プール

弾や火花を毎回newせず、消えたものを戻します。数が多く、寿命が短いものに向いています。

03

空間グリッド

近くの9マスだけを候補にします。全員対全員より、詳しい判定をずっと少なくできます。

$ go test -bench . -benchmem ./internal/perflab
BenchmarkCullReuse 5527 ns/op 0 allocs/op
BenchmarkGridNearby 1707 ns/op 2 allocs/op先に数字、次に改善
