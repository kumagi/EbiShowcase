編集する入口

games/tracks/tower-defense/tower-economy/main.go

この短い入口が、実際に学習者が開いて変更する Go ファイルです。

package main

import "github.com/kumagi/EbiShowcase/internal/towerdefenseplay"

func main() { towerdefenseplay.Run(towerdefenseplay.Config{Step: 5, Title: "TOWER ECONOMY"}) }

仕組みの抜粋

internal/towerdefense (shared engine)

上の games/tracks/tower-defense/tower-economy/main.go は開始設定を書く短いファイルです。移動・判定・得点などの処理はこのファイルにあり、次の抜粋で設定がどのルールへ渡るかを追えます。

// shared engine: Update advances enemies, chooses targets, and applies hits
