道を読み、塔を組み合わせる

タワーディフェンス

決められた道を進む敵から真珠を守ります。経路、射程、標的、弾、配置、ウェーブ、強化を一つずつ作ります。

LEVEL 01のUpdate / Drawを土台に、純粋な経路ロジックとEbitengineの操作・描画を重ねます。

LEARNING PATH

道の一本線から、2マップの防衛戦へ。

各ステップは単独で遊べます。完成版では6ウェーブとボスに挑戦します。

01

道しるべに沿って進む

曲がり角の座標を順につなぎ、進んだ距離から敵の今いる場所を計算します。

ウェイポイントと経路補間
→ 02

塔の射程を円で見る

塔と敵の距離を測り、円の半径以内に入ったかを見える形で確認します。

2点間距離と射程
→ 03

いちばん先の敵を選ぶ

射程内の敵から、ゴールへ最も近い一体を選び、黄色い輪で示します。

progressを比べる標的選択
→ 04

弾を飛ばして命中させる

標的へ少しずつ近づく弾を作り、接触した瞬間だけHPと演出を変えます。

弾の追跡・ダメージ・命中演出
→ 05

通貨で塔を配置する

敵を倒してコインを受け取り、道をふさがない場所へ塔を追加します。

配置条件と通貨ループ
→ 06

ウェーブをデータで増やす

出現数、間隔、HP、速さをウェーブ番号から読み、3回の攻防を作ります。

敵を時間差で出すウェーブ監督
→ 07

強化・鈍足・範囲攻撃

単体弾、敵を遅くする塔、近くをまとめて攻撃する塔を選び、既存の塔も3段階に強化します。

役割の違う塔とアップグレード
→ 08

Ebi Pearl Defense

2つの経路、6ウェーブ、塔3種類、強化、通貨、最終ボスをつなぎ、真珠の門を守ります。

複数マップとボスを統合する完成版
