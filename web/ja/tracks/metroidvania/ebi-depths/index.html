コードを読む前の用語メモ

Updateは次の一コマの数字を決め、Drawはその数字を絵にします。これがEbitengineのゲームループです。 PLAYABLEは『このページで実際に操作できる』という印です。Updateは数字と状態を進め、Drawはその結果を絵にします。

海老・天次郎の見方
Updateが天次郎の数字を変え、Drawが次の一コマを描きます。

func (g *Game) Update() error { return nil }

func (g *Game)の(g *Game)は、この仕事がGameの値を読むための名札（レシーバー）です。
