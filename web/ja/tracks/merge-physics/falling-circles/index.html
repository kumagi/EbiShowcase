HOW IT WORKS

毎回同じ小さな時間だけ進める

EbitengineのUpdateは基本的に1秒60回です。毎回、縦速度へ同じ重力を足し、その速度を位置へ足します。床を越えたら位置を戻し、縦速度を反対向きに小さくします。

velocityY += gravity
positionY += velocityY
if positionY+radius > floor {
    positionY = floor-radius
    velocityY *= -bounce
}
