スイカゲーム型の物理パズルを作る

合体物理パズル

スイカゲームのように、同じ種類の実がぶつかると合体して大きくなるパズルを作ります。落下する円から始め、重力、衝突、円を押し合って重ならないようにする仕組み（めり込み修正）、反発、同種合体を順に実装します。

共通基礎の続きです。LEVEL 01 の Update / Draw の上に、このジャンルの部品を1ステップずつ足します。主人公はオリジナルの 海老・天次郎（えび・てんじろう） です。

SPECIALIZATION / VFX基礎は習得済み

演出を足す練習ではなく、
合体物理パズルのルールを作る。

新しい課題は円の衝突解決、めり込み修正、反発、重力、合体サイズ曲線です。

MOTION BASELINE

動いて見えることも完成条件

MergeResultを一回の事実にし、消えた二体のSnapshotから新しい一体への収束を描き、同時合体は予算つき列で処理します。

TEST SEAM

派手さを外してもルールは同じ

同じ初期状態とtick数で同じ合体結果になり、粒数を0にしても半径・得点・ゲームオーバーが一致すること。

LEARNING PATH

小さな仕組みから
完成形へ。

難しい部分を一度に作らず、単独で動くミニゲームへ分解します。各ステップで前のコードを再利用し、最後に統合します。

01

落下する円

大きさの違う円を落とし、端末の速度に左右されない更新を作ります。

円・重力・固定時間ステップ
→ 02

円どうしの衝突

2つの円の中心距離から重なりを調べ、押し戻す方向を求めます。

中心距離と接触法線
→ 03

積み上げと反発

重なった円を押し合って重ならない位置まで戻し、重さと弾みやすさ（反発係数）から新しい速度を計算します。

重なりを直す押し戻し（めり込み修正）と、勢いを与える計算（インパルス）
→ 04

安定する物理世界

円どうしが重ならなくなるまで、押し戻す処理を何度も少しずつ繰り返し（反復解法）、動きが止まった円を休ませることで、たくさんの円を安定して積みます。

何度も押し戻す解き方（反復解法）・休眠・近くだけ調べる工夫（空間分割）
→ 05

同じ実の合体

同じ種類が触れたら2個を消し、中心位置に次の大きさを生成します。

衝突イベントと置き換え
→ 06

合体が気持ちいい瞬間

合体した場所から脈動・粒子・揺れを始め、連続合体をコンボとして見せます。

衝突イベントから時間差の演出を始める
→ 07

重力が変わる3つの課題

目標段階と重力の違う3ステージをデータとして順番に読み込みます。

物理定数と目標をステージデータにする
→ 08

Ebi Merge

次の実、投入操作、得点、危険ラインを統合して物理パズルを完成させます。
