合体物理パズル

落下する円から始め、重力、衝突、めり込み修正、反発、同種合体を順に実装します。

共通基礎の続きです。LEVEL 01 の Update / Draw の上に、このジャンルの部品を1ステップずつ足します。主人公はオリジナルの 海老・天次郎 です。

LEARNING PATH

小さな仕組みから
完成形へ。

難しい部分を一度に作らず、単独で動くミニゲームへ分解します。各ステップで前のコードを再利用し、最後に統合します。

01

落下する円

大きさの違う円を落とし、端末の速度に左右されない更新を作ります。

円・重力・固定時間ステップ
→ 02

円どうしの衝突

2つの円の中心距離から重なりを調べ、押し戻す方向を求めます。

中心距離と接触法線
→ 03

積み上げと反発

円を重ならない位置へ戻し、質量と反発係数から速度を変えます。

めり込み修正とインパルス
→ 04

安定する物理世界

複数回の衝突解決と休眠処理で、たくさんの円を安定して積みます。

反復解法・休眠・空間分割
→ 05

同じ実の合体

同じ種類が触れたら2個を消し、中心位置に次の大きさを生成します。

衝突イベントと置き換え
