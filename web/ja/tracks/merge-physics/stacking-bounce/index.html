DEEP DIVE / IMPULSE

重い円と軽い円は、同じように跳ねる？

大きな円ほど質量を大きくします。衝突の勢いを表すインパルスを質量に合わせて分けると、軽い円が多く動きます。

1

めり込み修正

重なった距離を質量に合わせて分けます。

overlap := radii-distance
2

質量

半径の2乗を円の重さとして使います。

mass = radius * radius
3

インパルス

接触方向の速度だけを反発させます。

velocity += impulse / mass

IN THE REAL GAME

位置を直してから速度を変える

まず重なりをなくし、近づいている時だけ反発を加えます。

correctPenetration(a, b) / relative := dot(b.velocity-a.velocity, normal) / if relative < 0 { applyImpulse(a, b, normal) }

WHY CORRECT FIRST?

沈み込みを防ぐ

速度だけ変えても重なった円は残ります。

WHY MASS?

大きさで動き方が変わる

軽い円は重い円より大きく押されます。
