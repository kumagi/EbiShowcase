THE ONE-LINE RULE

distance < a.r + b.r なら 法線方向へ半分ずつ戻す

片方だけ動かすと相手が「壁」みたいになります。半分ずつ戻すと、どちらも同じように跳ねます。

IN THE REAL GAME

重なりを半分ずつ戻す

Update の中で距離を測り、重なっていたら法線方向へ押し戻し、相対速度から跳ね返しを足します。vn := rvx*nx + rvy*ny は法線方向への内積（dot）です。

dx, dy := b.x-a.x, b.y-a.y
distance := math.Hypot(dx, dy)
if distance < a.r+b.r {
    nx, ny := dx/distance, dy/distance
    overlap := a.r + b.r - distance
    a.x -= nx * overlap / 2
    b.x += nx * overlap / 2
    a.y -= ny * overlap / 2
    b.y += ny * overlap / 2
    // 相対速度を法線へ射影して跳ね返す
    rvx, rvy := a.vx-b.vx, a.vy-b.vy
    vn := rvx*nx + rvy*ny
    if vn < 0 { // 近づいているときだけ
        a.vx -= vn * nx * 0.9
        a.vy -= vn * ny * 0.9
        b.vx += vn * nx * 0.9
        b.vy += vn * ny * 0.9
    }
}

WHY DISTANCE?

円は向きを選ばない

どこから触れても同じ式で判定できます。

WHY NORMAL?

最短方向へ戻せる

斜め衝突も、中心を結ぶ向きが一番短い抜け道です。
