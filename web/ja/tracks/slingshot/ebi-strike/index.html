DEEP DIVE / CONTACT PIPELINE

同じ「ぶつかった」を、相手ごとのイベントへ変える。

動いている味方は毎フレーム、壁・岩・敵・味方の順に接触を調べます。円を押し戻して反射したあと、相手が敵なら直接ダメージ、味方なら範囲波という別の結果を起こします。止まるまで同じ手番です。

1

移動して接触を解決

速度で位置を進め、重なった円を接する場所まで戻します。速度の法線方向を反転すれば、斜めにも跳ね返ります。

move(); bounceWalls(); reflectCircle()
2

相手から効果を選ぶ

敵ならその1体のHPを1減らします。味方ならその味方を中心に165px以内の敵へ波のダメージを送ります。

contactEnemy() / contactAlly()
3

摩擦で止まり手番交代

接触処理のあと速度へ0.982を掛けます。速さが0.11未満なら0へそろえ、activeをもう一人へ切り替えます。

if stopped { active = 1 - active }

IN THE REAL GAME

移動中と狙い中を混ぜない

movingがtrueの間だけ物理と接触を進めます。停止した一瞬に手番を変え、その後で次のドラッグ入力を受け付けます。

active.pos = active.pos.add(active.velocity)
bounceWalls(&active)
hitEnemies(&active)
hitAlly(&active)
active.velocity = active.velocity.scale(0.982)

if active.velocity.length() < stopSpeed {
    active.velocity = vec{}
    moving = false
    activeIndex = 1 - activeIndex
}

WHY COOLDOWN?

1回の接触を1回だけ数える

円が数フレーム触れ続けても、敵ごとの短い待ち時間によりHPが一気に0になりません。

WHY ONE ALLY WAVE?

一手の選択をはっきりさせる

味方効果は1ショットに1回。何度も同じ二人の間で跳ねて無限攻撃になるのを防ぎます。
