指で引いた方向と距離から矢印を描き、反対方向へ速度を与えます。

このステップで学ぶこと ドラッグベクトルと発射速度

これは共通基礎の続きです。まだゲームループを知らない人は、先にLEVEL 01 の Update / Drawを見てください。ここでは入力から速度を作るところへ進みます。

PLAYABLE / WEBASSEMBLY

真珠を引っぱり、3つのリングへ届けよう。

真珠からドラッグして離します。矢印キーで引く向きを変え、Spaceで発射することもできます。6発以内でクリア。

● LIVE / GO + WASMSTEP 01

DEEP DIVE / OPPOSITE VECTOR

引いた矢印を、マイナスでひっくり返す。

真珠の中心から指までの差をドラッグベクトルとして保存します。離した瞬間、XとYの両方にマイナスを付けると、引っぱった向きと正反対の発射速度になります。

1

2点の差を取る

指の座標から真珠の座標を引きます。右へ引けばdragXは正、下へ引けばdragYは正です。

dragX = pointerX - ballX
2

長すぎる引きを制限

長さが160を超えたら、160/length を掛けて同じ向きのまま縮めます（長さの比でスケール）。例: 長さ320なら半分にして最大160にそろえます。

dx *= 160 / length
3

反対向きへ発射

指を離した時だけ、dragXとdragYをマイナスにして速度へ変えます。下へ引いた真珠は上へ飛びます。

velocity = -drag * power

TRY IT / VECTOR LENGTH

向きの組み合わせで長さはどう変わる？

横だけのベクトルと斜めのベクトルを比べます。ゲームでは長さを測って最大値をそろえ、引きすぎを防ぎます。

TRY IT · GO

dx, dy := pointerX-ballX, pointerY-ballY
length := math.Hypot(dx, dy)
if length > 160 { dx *= 160/length; dy *= 160/length }

IN THE REAL GAME

離した1回だけ速度を決める

引いている間は照準を更新し、リリースされたフレームで速度へ変換します。

dragX := pointerX - ballX
dragY := pointerY - ballY
if released {
    vx = -dragX * 0.092
    vy = -dragY * 0.092
    moving = true
}

WHY OPPOSITE?

ゴムを戻す向き

後ろへ引くほど、元の場所へ戻ろうとする向きに強く飛びます。

WHY ON RELEASE?

照準と飛行を分ける

引いている途中で発射せず、決めた向きが離した瞬間に確定します。
