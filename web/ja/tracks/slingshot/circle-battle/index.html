DEEP DIVE / CIRCLE CONTACT

中心の距離を測り、重なった分だけ外へ戻す。

2つの円は、中心間の距離が半径の合計より短いと重なっています。判定したら真珠を円のふちまで押し戻し、十分な速さで初めてダメージを与えます。

1

円の重なりを探す

XとYの差から中心間の距離を計算します。半径20と30なら、距離50未満が接触です。

distance < playerR + enemyR
2

ふちまで押し戻す

敵から真珠へ向かう方向を長さ1にそろえ、半径の合計だけ離れた位置へ置きます。

player = enemy + normal * minDistance

if cooldown == 0 { hp--; cooldown = 24 }

IN THE REAL GAME

判定、位置修正、攻撃の順

先に重なりをなくし、接触方向の速さが1.5以上でcooldownが0の時だけ攻撃します。

if distance < playerR+enemyR {
    normal := delta / distance
    player = enemy + normal*(playerR+enemyR)
    impact := abs(dot(velocity, normal))
    if cooldown == 0 && impact >= 1.5 {
        hp--; cooldown = 24
    }
}

WHY PUSH OUT?

円がくっつかない

速度だけ反転しても重なりが残ると、次のtickでも衝突します。

WHY SPEED?

そっと触るのと攻撃を分ける

接触方向の速さを使えば、勢いをつける遊びになります。
