モンスターストライク型ゲームを作る

ひっぱり反射アクション

モンスターストライクのような、キャラクターを引っぱって発射する対戦アクションを作ります。引っぱる照準から、円の反射、減速、敵への接触攻撃、味方に当てて発動する「友情効果」へ進みます。

はじめて読む人へ

このコースの組み立て図

引く→速度→反射→命中の順に、海老・天次郎（えび・てんじろう）と一部品ずつ作ります。 このページの1本はコース全体で共通するgame状態の境界です。ルールはUpdateへ足し、Drawの順番や見た目は自由に差し替えられます。下の01から順に、その中へ各仕組みを実装していきます。

このページの1本
説明と次のコードを対応させてから、ゲームで変化を確かめよう。

func (g *Game) Update() error { return nil }
01 引く→02 速度→03 反射→04 命中

共通基礎の続きです。LEVEL 01 の Update / Draw の上に、このジャンルの部品を1ステップずつ足します。主人公が登場するステップではオリジナルの 海老・天次郎（えび・てんじろう） を使います。

LEARNING PATH

小さなルールを
つないで完成へ。

各ステップを単独で遊べる大きさにし、アルゴリズムを目で確認してから次の仕組みへ進みます。

01

ひっぱって発射

指で引いた方向と距離（ドラッグベクトル）から矢印を描き、引いた方向の反対へ速度を与えて発射します。

指で引いた方向と距離（ドラッグベクトル）から発射速度を決める
→ 02

壁で跳ね返るボール

壁の向きに対して鏡のように速度を反射させ（法線反射）、跳ねるたびに少しだけ勢いを失わせます（反発係数でどれくらい弾むか決める）。

壁で跳ね返る仕組み（法線反射）と、弾みやすさ（反発係数）
→ 03

円形キャラクターの衝突

敵へ触れた速さと回数を判定し、同じ敵への連続ヒットを制御します。

円衝突と接触ダメージ
→ 04

減速して止まる

速度を少しずつ減らし、十分遅くなったら正確に停止して次の手番へ進みます。

摩擦・停止しきい値・ターン終了
→ 05

味方に当てる友情効果

味方へ触れた瞬間にレーザーや回復など、データで選んだ「友情効果」を発動します。

味方への接触で発動する友情効果
→ 06

点で見える発射予測

引っぱった速度を数tickだけ先に計算し、白い点として発射前に表示します。

同じ速度式を予測描画にも使う
→ 07

反射を変える3つのステージ

柱配置・敵数・必要ヒット数の異なる3面をステージデータから読み込みます。

同じ反射ルールへ違う盤面を渡す
→ 08

Ebi Strike

複数キャラクターの手番、必殺技、敵AI、障害物を統合して完成させます。

手番・必殺技・ステージギミック
