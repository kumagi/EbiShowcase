減速して止まる

速度へ摩擦の数を毎フレーム掛け、十分遅くなった瞬間に速度を0へそろえます。

このステップで学ぶこと速度減衰・停止しきい値・手番終了

PLAYABLE / WEBASSEMBLY

6投以内に3つの島へぴたりと止めよう。

赤い玉から引っぱって離します。ICE / NORMAL / SANDと停止値を変え、止まる距離を予想してください。PCは1〜3と-/+でも調整できます。

● LIVE / GO + WASMSTEP 04

DEEP DIVE / DAMP AND SNAP

少しずつ弱め、最後はきっぱり0にする。

摩擦を0.978にすると、速さは前のコマの97.8%になります。ずっと掛けても数字は0に近づくだけなので、停止しきい値より小さくなったら0を代入し、手番を終わらせます。

1

位置を速度だけ進める

まず今の速度で位置を変えます。壁に当たれば向きを反転してから、次の計算へ進みます。

pos.x += velocity.x
2

速度へ摩擦を掛ける

0.990は長く滑り、0.960はすぐ減速します。xとyへ同じ数を掛ければ向きを保ったまま弱くなります。

velocity.x *= friction
3

小さければ正確に止める

速さがしきい値未満ならxもyも0にします。ここで着地点を調べ、次の手番へ進めます。

if speed < stop { velocity = vec{} }

1コマずつ見て、停止を決めよう

進めるたびに位置が変わります。「もう動いていないと見なせる」と思ったところで停止してください。ゲームではこの判断を速さの数で自動化します。

IN THE REAL GAME

順番は「移動→摩擦→停止判定」

停止判定を摩擦より後に置くので、このフレームで小さくなった速度もすぐ0になります。

pos.x += velocity.x
pos.y += velocity.y
velocity.x *= friction
velocity.y *= friction

if math.Hypot(velocity.x, velocity.y) < stopThreshold {
    velocity = vec{}
    finishShot()
}

WHY MULTIPLY?

速いほど大きく減る

同じ数を引くより、割合で減らすほうが最初は勢いよく、最後はゆっくり止まる動きになります。

WHY A THRESHOLD?

終わらない手番を防ぐ

小数の速度は0になりきらないため、「ほとんど停止」を数字で決める必要があります。
