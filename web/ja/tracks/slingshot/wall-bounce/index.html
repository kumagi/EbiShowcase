DEEP DIVE / WALL NORMAL

壁へ向かう速さだけ、反対向きにする。

速度は横のvxと縦のvyに分かれています。縦の壁へ当たったら横向きだけ、床や天井なら縦向きだけを反転します。壁からまっすぐ外へ向く矢印を法線といい、反射は法線方向の速度をひっくり返す計算です。

1

半径まで含めて接触

円の中心が壁へ着いてからでは半分めり込みます。中心から半径を引いた左端が壁を越えたら接触です。

if x-radius < leftWall
2

片方の速度を反転

左右の壁はvxの符号だけを変えます。vyを残すので、入ってきた角度と同じ角度で出ていきます。

vx = -vx
3

少し勢いを失う

反転した速さへ0.92を掛けます。この反発係数が1なら同じ速さ、0に近いほど壁へ勢いを渡します。

vx = -vx * restitution

IN THE REAL GAME

位置を壁の内側へ戻してから反転

めり込んだままだと次のフレームでも接触し、何度も反転します。まず円を境界へ戻し、必ず内向きの速度へ直します。

if x-radius < leftWall {
    x = leftWall + radius
    vx = math.Abs(vx) * restitution
    bounces++
}
if y-radius < topWall {
    y = topWall + radius
    vy = math.Abs(vy) * restitution
    bounces++
}

WHY CLAMP?

壁の中に残さない

位置を境界へ戻すことで、1回の衝突を1回だけ数えられます。

WHY TWO PARTS?

角度を保てる

壁と平行な速度は変えないので、鏡で光が反射するような軌道になります。

TUNE IT

反発係数を変える

0.5と1.0を試し、3回目の反射後に進める距離を比べよう。
