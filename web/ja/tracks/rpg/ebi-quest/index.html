HOW IT WORKS

クエスト番号が物語のしおりになる

questが0なら仲間探し、1なら結晶、2ならボス、3なら帰還です。番号が変わるたび位置・HPと一緒にlocalStorageへ保存し、ページを閉じても続きから始めます。

type SaveData struct { X, Y, HP, Quest int }
quest++
save(SaveData{player.X, player.Y, player.HP, quest})
