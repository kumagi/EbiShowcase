THE DIALOGUE RULE

front := point{g.p.x + g.facing.x, g.p.y + g.facing.y} 正面にNPCがいる？ who, ok := g.npcs[front] フラグで分岐してセリフを設定 if !g.quest { g.quest = true; g.talk = [...] }

フラグは「出来事が起きた」という事実を記録します。同じキャラに何度でも話しかけられるRPGを作るとき、フラグなしでは毎回同じセリフになってしまいます。

IN THE REAL GAME

interact() — フラグを確認してセリフを決める

スペースキーや画面上部タップで interact() が呼ばれます。正面のNPCを調べ、長老なら quest と herb の状態に応じてセリフを切り替えます。

func (g *game) interact() {
    front := point{g.p.x + g.facing.x, g.p.y + g.facing.y}
    who, ok := g.npcs[front]
    if !ok { return }
    switch who {
    case "elder":
        if !g.quest {
            g.quest = true
            g.talk = []string{"Elder: A healing herb grows northwest.",
                              "Please bring it back to me!"}
        } else if !g.herb {
            g.talk = []string{"Elder: The herb is in the northwest corner."}
        } else {
            g.talk = []string{"Elder: You found it! Thank you, hero!",
                              "QUEST COMPLETE"}
            g.clear = true
        }
    }
}

WITHOUT FLAGS

フラグなしだと？

状態を記録しないと、何度話しかけても同じセリフが出ます。「薬草を持ってきた後」も「最初に話した時」も区別できません。

WITH FLAGS

bool 1個でストーリーが分岐する

2つのフラグで長老のセリフが3段階に変わります。フラグを増やせば、クエストの複雑さをいくらでも広げられます。
