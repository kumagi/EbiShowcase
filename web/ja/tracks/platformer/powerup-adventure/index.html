HOW IT WORKS

ゲームは小さな仕組みの組み合わせ

移動、ジャンプ、敵、コイン、変身は別々の小さな規則です。Updateで順番に呼び、ゴールしたらstage番号を増やしてデータを読み直すと、1本の冒険になります。

updateMovement()
updateCollectibles()
updateEnemies()
if reachedGoal {
    stage++
    loadStage(stage)
}

GAME + LESSON + CHALLENGE

部品をつないで作品にする

01

集める

コインの取得でスコアを増やし、配列から消します。

02

変身する

パワーアップ取得中は体を大きくし、1回のダメージを防ぎます。

03

次へ進む

ゴールでステージ番号を変え、次のステージを初期化します。
