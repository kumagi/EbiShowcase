HOW IT WORKS

地面にいる時だけジャンプする

床へ上から着地した時に grounded をtrueにします。ジャンプは grounded の時だけ許可し、空中で何度も跳べないようにします。

if hitFloor { grounded = true }
if jumpPressed && grounded {
    velocityY = -jumpPower
}
