矢印キー/WASD、または画面下のボタンで移動・旋回します。最終章ではSPACEかRAYボタンで射撃します。

DEEP DIVE

割り算で遠近感を作る

同じ大きさの物は、近いほど大きく見えます。画面の高さを距離で割ると、近い壁ほど高い縦帯になります。一本の光線が、FPS画面の一本の列へ変わりました。

MAP

1

Update が担当

キー・タッチを読み、位置・HP・ゲージ・タイマー・盤面を変えます。判定やキュー操作もこちらです。

入力 → ルール → game を更新
2

Draw が担当

できあがった game の値を読んで、絵・文字・エフェクトを置きます。game の値は変えません。

game を読む → ピクセルへ投影

このページのルールは Update 側（または Update が呼ぶ純粋な関数）へ置き、Draw はその結果を表示するだけにします。

編集する入口

games/tracks/raycaster/distance-strip/main.go

この短い入口が、実際に学習者が開いて変更する Go ファイルです。

package main
