2Dの数学で奥行きを描く

2Dから擬似3Dへ：レイキャスト迷路FPS

Ebitengineの2D描画だけで、Doomのように見える迷路を組み立てます。世界は2Dマップのまま。光線の距離を縦帯の高さへ変え、最後は鍵と敵のいる脱出ゲームへ育てます。

SPECIALIZATION / VFX基礎は習得済み

演出を足す練習ではなく、
レイキャスト迷路FPSのルールを作る。

新しい課題はDDA、壁投影、魚眼補正、深度、敵スプライトと射撃です。

A04物理から導く姿勢A05上限つき描画履歴A11ヒットリアクション時間軸

MOTION BASELINE

動いて見えることも完成条件

速度から武器・歩行bobを導出し、弾道履歴を制限し、命中と被弾をリアクション時間軸にします。

TEST SEAM

派手さを外してもルールは同じ

光線距離と遮蔽を純粋に試し、銃の反動や画面フラッシュで命中距離が変わらないこと。

マップ、角度、当たり判定、カメラを統合する、応用編の発展話題です。

EXPERIMENTAL LEARNING PATH

一本の光線から、遊べる迷路FPSへ。

Ebitengineの2D描画だけで、Doomのように見える迷路を組み立てます。世界は2Dマップのまま。光線の距離を縦帯の高さへ変え、最後は鍵と敵のいる脱出ゲームへ育てます。

01

向きを持って迷路を歩く

2Dマップの中で左右に向きを変え、向いている方向へ進みます。

角度から進む量を作る
→02

一本の光線を壁まで飛ばす

プレイヤーの正面へ光線を伸ばし、最初にぶつかる壁と距離を探します。

DDAでマス目を一つずつ渡る
→03

距離を壁の高さへ変える

近い壁を高い縦帯、遠い壁を低い縦帯として描きます。

割り算で遠近感を作る
→04

画面の列ごとに光線を飛ばす

視野を扇のように分け、たくさんの縦帯を横へ並べて擬似3D迷路を作ります。

2Dの距離を3Dらしい画面へ投影する
→05

壁模様と魚眼補正を足す

壁の当たり位置から模様を選び、端が曲がって見える魚眼のゆがみを直します。

当たり位置と補正距離
→06

3ミッションのEbi Raycasterを完成させる

敵・鍵・出口にHPとダメージ、評価、BESTを組み合わせ、3つの脱出ミッションを遊べる迷路FPSにします。

ミッションデータと深度をゲームルールへつなぐ
