HOW IT WORKS

小さな勝負をラウンドで包む

各ラウンドは両者のHPと45秒を初期化します。HPが0か時間切れになったら残りHPを比べて勝者へ1点。どちらかが2点になれば試合終了です。

if hp == 0 || timer == 0 {
    roundWins[winner]++
    if roundWins[winner] == 2 { matchOver = true }
    else { resetRound() }
}
