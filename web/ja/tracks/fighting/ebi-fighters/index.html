IN THE REAL GAME

ラウンド結果から勝敗を積む

HP比較で勝者を決め、2勝でマッチ終了、さもなくば次ラウンドへ。

if g.p.hp > g.ai.hp {
    g.pWins++
} else {
    g.aiWins++
}
if g.pWins >= 2 || g.aiWins >= 2 {
    g.matchOver = true
} else {
    g.round++
    g.resetRound()
}
