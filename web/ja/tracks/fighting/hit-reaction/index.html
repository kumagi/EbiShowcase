HOW IT WORKS

「止まる時間」と「動けない時間」は別

ヒットストップ中はゲーム全体のUpdateをすぐ終えます。その後のけぞり時間だけ相手の行動を止め、ノックバック速度を位置へ足して後ろへ動かします。

if hitStop > 0 { hitStop--; return nil }
if hitStun > 0 {
    enemy.X += knockbackVelocity
    hitStun--
}

GAME + LESSON

命中の手ざわりを3つに分ける

01

止める

短い停止で命中を強く見せます。

02

封じる

のけぞり中は相手の入力を受けません。

03

押す

速度を減らしながら相手を後退させます。
