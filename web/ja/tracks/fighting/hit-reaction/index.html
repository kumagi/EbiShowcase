DEEP DIVE / HIT REACTION

当たった瞬間は、
なぜ重く見えるの？

答えはヒットストップです。当たったフレームだけ世界を止めると、脳が「いま当たった！」と強く覚えます。そのあとヒットスタンで相手を操作不能にし、ノックバックで後ろへ流します。

1

ヒットストップ

当たった瞬間、数フレームだけ Update を早期 return します。背景色を変えると「止まった」が目にもわかります。

if hitstop > 0 { hitstop--; return nil }
2

ヒットスタン

スタン中は相手の通常AIを止め、のけぞり専用の動きだけ許可します。

if stun > 0 { stun--; p2 += v2 }
3

ノックバック

押し出し速度 v2 を与え、毎フレーム減衰させます。だんだん止まるので自然です。

v2 = 7; v2 *= .86

TRY IT / LAB

ヒットストップとのけぞり

ヒットで世界を止め、のけぞり中は速度を減衰させます。

本物のゲームの公式を、手でゆっくり回す模型です。

stop → stun → v*=0.86
hitstop0
stun0
v20.00

THE REACTION CHAIN

hitstop = 8 つぎに stun = 25; v2 = 7

順番が大切です。まず世界を止め、次にスタンと押し出しをセットします。止める時間を長くすると重く、短くすると軽快になります。

IN THE REAL GAME

当たったフレームで
3つを一気にセット

攻撃のアクティブフレームで距離を見て、ヒットならストップ・スタン・速度を入れます。

if g.attack == 12 && g.p2-g.p1 < 115 {
    g.hitstop = 8
    g.stun = 25
    g.v2 = 7
    g.hits++
}

WITHOUT HITSTOP

当たった感じが薄い

ダメージは入っても、画面が止まらないと「かすった」ように見えます。

WITH REACTION

コンボの余白が生まれる

相手が動けない間に次の入力が間に合い、コンボへつながります。

3つの数字をいじろう

hitstop・stun・v2 を変えるだけで、軽いパンチにも重い一撃にもなります。
