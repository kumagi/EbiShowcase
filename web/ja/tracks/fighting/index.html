判定矩形から対戦格闘を作る

2D格闘ゲーム

ゲームセンターにあるような、1対1で技を出し合う対戦格闘ゲームを作ります。キャラクター同士がぶつからないための箱（押し合い判定）、攻撃が当たる場所を示す箱（喰らい判定・攻撃判定）から始め、技のフレーム、当たった瞬間だけ時間を止める「ヒットストップ」、コンボへ進みます。

共通基礎の続きです。LEVEL 01 の Update / Draw の上に、このジャンルの部品を1ステップずつ足します。主人公はオリジナルの 海老・天次郎（えび・てんじろう） です。

はじめて読む人へ

このコースの組み立て図

攻撃矩形→フレーム→反応→防御→連続技の順に、海老・天次郎（えび・てんじろう）と一部品ずつ作ります。 このページの1本はコース全体で共通するgame状態の境界です。ルールはUpdateへ足し、Drawの順番や見た目は自由に差し替えられます。下の01から順に、その中へ各仕組みを実装していきます。

このページの1本
説明と次のコードを対応させてから、ゲームで変化を確かめよう。

func (g *Game) Update() error { return nil }
