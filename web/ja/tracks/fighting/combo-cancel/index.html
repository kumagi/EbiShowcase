HOW IT WORKS

早すぎた入力を8フレームだけ覚える

ボタンを押すとbufferへ技を入れ、毎フレーム残り時間を減らします。今の技が当たり、キャンセル可能な時間になったら、保存した次の技をすぐ始めます。

if pressed { buffer = nextMove; bufferLife = 8 }
if hit && canCancel && bufferLife > 0 {
    start(buffer)
}
