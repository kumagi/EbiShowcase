HOW IT WORKS

重なった判定を決まった順に調べる

ガードは打撃に勝ち、投げはガードに勝ち、打撃は投げの準備に勝ちます。複数の条件が同時に起きても、優先順位表を1か所に置けば結果がぶれません。

func wins(a, b Action) bool {
    return a==Guard && b==Strike ||
           a==Throw && b==Guard ||
           a==Strike && b==Throw
}
