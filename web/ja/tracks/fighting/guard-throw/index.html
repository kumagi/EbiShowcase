DEEP DIVE / RPS PRIORITY

ガードは万能？
投げはどう勝つ？

ガードは打撃に勝ち、投げはガードに勝ち、打撃は投げに勝ちます。勝ち負けを関数1つにまとめると、あとから技を足してもルールが崩れません。

1

3つの選択肢

strike / guard / throw を定数で表します。入力も同じ番号にそろえます。

const ( strike, guard, throw )
2

勝ち表

組み合わせごとの勝敗を関数に書きます。同じ技どうしは相打ちです。

wins(a,b) bool

THE RPS RULE

GUARD > STRIKE かつ THROW > GUARD > …

サイクルにすると穴がなくなります。どれか1つだけ強い技を作ると、読み合いが死にます。

IN THE REAL GAME

勝ち判定は
関数に閉じ込める

入力を受け取ったら wins で判定し、得点かダメージを分けます。

func wins(a, b int) bool {
    return (a == strike && b == throw) ||
           (a == throw && b == guard) ||
           (a == guard && b == strike)
}
