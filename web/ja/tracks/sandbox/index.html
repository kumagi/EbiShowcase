掘って作れる世界を作る

2Dブロックサンドボックス

マインクラフトのような、ブロックを置いたり壊したりして世界を作るゲームを作ります。1個のブロックを置くところから、世界を区切る「チャンク」、クラフト、生き物、ワールド保存へ広げます。

はじめて読む人へ

このコースの組み立て図

座標変換→壊す/置く→持ち物→生成→区画の順に、海老・天次郎（えび・てんじろう）と一部品ずつ作ります。 このページの1本はコース全体で共通するgame状態の境界です。ルールはUpdateへ足し、Drawの順番や見た目は自由に差し替えられます。下の01から順に、その中へ各仕組みを実装していきます。

このページの1本
説明と次のコードを対応させてから、ゲームで変化を確かめよう。

func (g *Game) Update() error { return nil }
01 座標変換→02 壊す/置く→03 持ち物→04 生成→05 区画

共通基礎の続きです。LEVEL 01 の Update / Draw の上に、このジャンルの部品を1ステップずつ足します。主人公が登場するステップではオリジナルの 海老・天次郎（えび・てんじろう） を使います。

LEARNING PATH

ひとつの仕組みから、
遊べる世界へ。

データの変化とアルゴリズムを小さなゲームで確かめ、前のステップを再利用しながら完成形へ進みます。

01

置く・壊す

指したマスの種類を書き換え、ブロックの設置と破壊を作ります。

変更できるタイルマップ
→ 02

地形を生み出す

同じシード（種になる数字）なら同じ地形になる、土・石・空洞の世界を、なめらかにでこぼこするランダム（ノイズ）から生成します。

でこぼこを作るランダム（ノイズ）・地層・シード
→ 03

チャンクで広い世界

世界を「チャンク」という小さな正方形の区画に分け、プレイヤーの近くだけ更新・描画します（見えない部分を描かない工夫＝カリング）。

チャンクに分けての読み込みと、見えない部分を描かない工夫（カリング）
→ 04

道具袋とクラフト

集めたブロックを束ね、材料の組み合わせから新しい道具を作ります。

スタック・レシピ・消費
→ 05

道具と明かり

道具ごとの得意素材と、松明から広がる簡易的な明るさを計算します。

採掘速度とタイル光源
→ 06

生き物のいる世界

時間帯と地形を条件に生き物を出現（スポーン）させ、歩行や追跡をさせます。

生き物が現れる仕組み（スポーン）・昼夜・簡単なAI
→ 07

ブロックが壊れる一瞬を作る

道具を振りかぶり、当て、破片が飛び、元の姿勢へ戻るまでを時間に分けます。

接触tickで結果を一度だけ決める
→ 08

島データから3つの冒険を作る

地形、資源、敵、必要な明かりをデータにまとめ、同じルールで違う島を順番に遊びます。

クリア条件をデータから組み立てる
→ 09

Ebi Craft Expedition

苔の野営地、結晶洞窟、火の島を巡り、採取・道具・建築・敵を一つの冒険へつなぎます。

合計得点とベスト記録で別の採取順を試す
