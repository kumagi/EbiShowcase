DEEP DIVE / MUTABLE TILE MAP

マスの番号を書き換えると、世界の形が変わる。

盤面は board[y][x] という数字の表です。0は空気、1は土、2は石。置く時は在庫を1減らして番号を書き、壊す時は番号を0へ戻して素材を在庫へ返します。

1

画面をマス番号へ

押した座標から盤面左上を引き、1マスの幅で割ります。これで列xと行yが決まります。

作業者から3歩以内で、選んだ素材が1個以上の時だけ置けます。失敗した操作ではデータを変えません。

if inReach(x,y) && stock[kind] > 0

IN THE REAL GAME

置く前に条件を全部確認する

空きマス、操作範囲、在庫の3条件を通った時だけ盤面と在庫を同じ操作で更新します。

if board[y][x] == empty &&
   inReach(x, y) && stock[selected] > 0 {
    board[y][x] = selected
    stock[selected]--
}
// break: inventoryへ戻す
stock[board[y][x]]++
board[y][x] = empty

WHY INVENTORY?

WHY RANGE?

どこでも変更させない

作業者の近くに限ると、位置と世界変更がつながります。
