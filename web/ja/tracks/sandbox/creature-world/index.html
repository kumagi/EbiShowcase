DEEP DIVE / CREATURE STATE MACHINE

「いつ・どこ・どうする」を別々の条件にする。

スポーンは生まれる条件、状態機械は生まれた後の行動です。昼の草地なら草食生物、夜の岩場なら追跡者を出します。その後は各creatureが持つstateを、距離やえさ、松明から切り替えます。

1

時間と地形でスポーン

一定時間ごとに昼夜を調べます。昼は左の草地、夜は右の岩場という条件を通った場所へだけ追加します。

if isDay() { spawnGrazer(meadow) }
2

状態ごとに行動を選ぶ

WANDERはランダム歩行。近づいた草食生物はCURIOUS、夜の生物はCHASE。松明を受けるとFLEEになります。

switch creature.state { ... }
3

行動から次の状態へ

CURIOUSへベリーを渡すとFRIENDになって追従します。CHASEに触られるとHPが減り、その生物はいったんFLEEします。

curious + feed → friend

IN THE REAL GAME

状態を変えてから、その状態の速度を決める

まず距離から遷移を判定し、その後に現在state用の方向と速さを使います。1フレームに古い行動と新しい行動を混ぜません。

distance := distance(creature.pos, player)
switch creature.state {
case wander:
    if distance < 115 {
        creature.state = curious
    }
case curious:
    creature.dir = direction(creature.pos, player)
    if distance > 170 {
        creature.state = wander
    }
}

WHY TWO DISTANCES?

境目でガタガタさせない

115で近づき、170で離れるようにすると、境目を少し動いただけで状態が毎コマ往復しません。

WHY STATE PER CREATURE?

同じ種類でも別行動

1匹はFRIEND、もう1匹はWANDERのように、各個体が自分の記憶を持てます。
