DEEP DIVE / COLUMN COMPACTION

落下は物理計算ではなく、列の中身を下から詰め直せる。

消すマスへ-1という空き値を入れます。各列を下から読み、空きでないピースだけをwrite位置へ書くと、順番を保ったまま下へ集まります。

1

消す代わりに空きを置く

配列の大きさは変えず、消えたマスだけemptyにします。Drawはemptyなら輪郭だけ描きます。

board[y][x] = empty
2

書き込み位置を下から動かす

writeを一番下に置き、実物を見つけるたびそこへ移してwriteを1上げます。

board[write][x] = board[y][x]
3

残った上を補充

詰め終わったあともemptyのマスへ新しい種類番号を入れます。これで盤面サイズが元へ戻ります。

if cell == empty { cell = randomPiece() }

IN THE REAL GAME

1列を下から圧縮する

readとwriteを分けると、穴が何個続いても同じループで処理できます。

write := rows - 1
for read := rows-1; read >= 0; read-- {
    if board[read][x] != empty {
        board[write][x] = board[read][x]
        write--
    }
}

WHY EMPTY?

途中の状態を表せる

消去直後、落下後、補充後を同じ配列の値だけで区別できます。

WHY COLUMNS?

重力方向と一致する

列ごとなら左右へずらさず、真下だけへピースを集められます。
