特殊ピース工房

4個、5個、L字の形を見つけ、それぞれ違う範囲を消す特殊ピースへ変身させます。

このステップで学ぶこと形の判定と範囲効果

PLAYABLE / WEBASSEMBLY

3種類の特殊ピースを作って発動しよう。

上の形をタップするか1・2・3キーで選びます。下のボタンまたはSpaceで「作る→発動→次へ」。全部にOKが付けばクリアです。

● LIVE / GO + WASMSTEP 05

DEEP DIVE / DETECT → KIND → EFFECT

特殊ピースは「形の判定」と「消す範囲」の2関数。

マッチ3の特殊ピースは魔法ではありません。①盤面から形を読み取る（4連・5連・L/T）、②種類ごとに消すマスの集合を返す、の2段です。効果の違いは「集合の作り方」だけで、最後の消去処理はふつうのマッチと同じにできます。

1

長い形から判定する

同じ色の連続を数えます。横5→色消し、横4→列ロケット。L/Tは「横も縦も3以上」を直線より先に調べます。

if h>=3 && v>=3 { kind = AreaBomb }
2

種類をマスに置く

判定の中心マスへ special を書き込みます。見た目のアイコンは kind のスイッチで描くだけです。

board[y][x] = special(kind)
3

効果は Set を返す

ロケットなら同じ行、色波なら同色すべて、範囲爆弾なら 3×3。返した Set をまとめて empty にします。

for p := range effect(kind, at)

消すマスの集合を先に作る

まず「4連／5連／L字」のどれかを選び、効果範囲の集合を集めます。爆弾もラインも、最後は同じ clear(Set) に渡せます。

効果ごとに描画を分けても、消去パイプは1本にまとめると安全です。

形の判定と範囲計算を1つの巨大ifにしないのが、特殊ピースを増やすコツです。

IN THE REAL GAME

detectSpecial と effectCells を分ける

デモはレシピ（4 / 5 / L）を選んでから FORGE→発動します。本番でも「作る瞬間」と「消す瞬間」を分けて書くと、連鎖で特殊が巻き込まれたときに同じ effectCells を再利用できます。

kind, at, ok := detectSpecial(board)
if !ok { return }
specials[at] = kind

// 発動時
for p := range effectCells(kind, at, board) {
    board[p.y][p.x] = empty
}

WHY ORDER?

Lを直線より先に

Lの腕は3連でもあるので、先に直線判定すると曲がった形を取りこぼします。

WHY A SET?

重なりを一度だけ

行消しと爆弾が同じマスを指しても、mapなら二重消去や二重得点になりません。
