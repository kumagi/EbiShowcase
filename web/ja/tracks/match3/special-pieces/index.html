DEEP DIVE / PATTERN TO EFFECT

同じ色の「数」と「曲がり方」を、技の設計図にする。

ふつうの3個並びより難しい形ができたら、ごほうびを作れます。まず盤面を調べて形の種類を決め、次に消す場所の集合を作る。この2段階に分けると、特殊ピースを増やしても整理できます。

1

縦と横の長さを数える

あるマスから同じ色が左右にいくつ、上下にいくつ続くかを数えます。5個なら色消し、4個なら一列消しの材料です。

if horizontal >= 5 { kind = ColorWave }
2

交差を先に調べる

横も縦も3個以上ならL字やT字の曲がった形です。4個並びより先に判定しないと、曲がりをただの直線とまちがえます。

if horizontal >= 3 && vertical >= 3
3

効果範囲を集合にする

一列、同じ色全部、中心の3×3など、消すマスをmapへ追加します。同じマスが重なっても一度だけ数えられます。

affected[point{x, y}] = true

「マスを追加」で爆発の候補を作り、「1マス処理」で順番に消します。重なる効果も、最後は処理するマスの集合になります。

IN THE REAL GAME

判定と効果を別の関数にする

detectSpecialは「何ができたか」を返し、activateは種類を見て「どこへ届くか」を決めます。一つの長いif文にしないのが大切です。

kind, center, ok := detectSpecial(board)
if ok {
    special.kind = kind
    special.pos = center
}

for p := range effectCells(special) {
    board[p.y][p.x] = Empty
}

WHY ORDER?

特別な形から判定

L字の一部には3個の直線もあります。いちばん条件が多い形から調べると、正しいごほうびを選べます。

WHY A SET?

重なりを安全にする

行消しと爆弾が同じマスへ届いても、mapなら二重に得点を足したり二度消したりしません。
