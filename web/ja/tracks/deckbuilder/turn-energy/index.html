PLAYABLE / WEBASSEMBLY

3エナジーを組み合わせて勝とう。

カードを1〜3で使用し、Eまたは下ボタンでターン終了。

● LIVE / GO + WASMSTEP 02

DEEP DIVE / TURN PHASE

どうして敵は、カードを選んでいる途中に攻撃しないの？

今のフェーズを1つだけ覚え、そのフェーズのUpdateだけを動かすからです。プレイヤー終了後に敵へ順番を渡します。

1

ターン開始

エナジーを3へ戻し、ブロックを準備します。

energy = 3
2

カード使用

コストを引き、残り範囲で何枚も使えます。

energy -= card.Cost
3

敵行動

終了ボタンでphaseをEnemyへ変えます。

phase = EnemyPhase

IN THE REAL GAME

フェーズでUpdateを分岐する

敵フェーズ中はカード入力を読まず、敵行動の完了後にエナジーを戻します。

switch phase { case PlayerPhase: playCardsOrEnd(); case EnemyPhase: enemyAct(); energy = 3; phase = PlayerPhase }
