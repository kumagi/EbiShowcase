DEEP DIVE / GRAPH MAP

マップは絵ではなく「地点」と「つながり」のデータ。

各地点をnodeとして作り、nextに次へ進める番号を保存します。現在地のnextだけ入力を受け付ければ、線がない場所へ飛ぶことはありません。

1

地点を作る

座標、種類、危険度、報酬を1つの構造体へ入れます。Drawは座標を、イベント処理は種類を読みます。

type node struct { x, y float32; kind string }
2

つながりを番号で持つ

nextが[3,4]なら次は3番か4番です。このつながり全体をグラフと呼びます。

next []int
3

訪問済みを覚える

選んだ地点をvisitedに記録し、通った道を色で見せます。セーブデータにも現在地と訪問履歴を保存できます。

visited[current] = true

IN THE REAL GAME

現在地から出る線だけを調べる

タップした丸がnextのどれかなら移動し、その地点のイベントを1回実行します。

options := nodes[current].next
for _, id := range options {
    if tapped(nodes[id]) {
        current = id
        resolve(nodes[id].kind)
    }
}
