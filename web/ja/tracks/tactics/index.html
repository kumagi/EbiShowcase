LEARNING PATH

小さな判断を重ねて、3つの任務へ。

移動コスト、武器射程、敵の狙い、任務データを小さなゲームで学び、3つの戦場を戦う完成版へ進みます。

01

地形コストで移動範囲を作る

平地1・森2・山3を足し、同じ移動力でも道によって届く場所が変わる盤面を作ります。

安い道から確定する重み付き探索
→ 02

剣と弓の射程を読み分ける

剣は1マス、弓は2マスという違いを盤面に重ね、武器を選んで正しい距離の敵を狙います。

マンハッタン距離と武器データ
→ 03

敵の狙いを見せる

敵が次に狙う仲間を先に表示し、攻撃が来る場所へガードを置く読み合いを作ります。

敵AIの決定と表示を同じデータにする
→ 04

任務をデータで3つ作る

開始地点、目的地、障害物、ターン制限を表にして、到達・救出・脱出の3任務を同じルールで再生します。

勝利条件とマップを任務データへまとめる
