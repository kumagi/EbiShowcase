迷路追跡ゲーム

パックマンのような、迷路の中でドットを食べながら敵から逃げるゲームを作ります。タイル上の移動から、次に曲がる方向を早めに覚えておく仕組み（先行入力）、交差点、敵のモード、経路選択、パワー状態へ進みます。

SPECIALIZATION / VFX基礎は習得済み

迷路追跡ゲームのルールを作る。

新しい課題はタイル移動、先行入力、交差点、敵モード、経路選択です。

少し早く押した方向を覚えておき（先行入力）、次の交差点で自動的に曲がります。

次の方向を早めに覚える仕組み（先行入力）と交差点判定
→ 03

ドットを食べる迷路

→ 04

巡回と追跡

敵モードと目標タイル
→ 05

交差点AI

敵が交差点に来たら、進める方向をすべて比べて「目標にいちばん近づく道」を選びます。たとえば右か下かで迷ったら、目標までの残り距離が短い方を選ぶ、という考え方です（距離評価）。

目標に近づく道を選ぶ考え方（距離評価）と逆走禁止
→ 06

敵を巡回・追跡・捜索に分け、主人公を見失っても最後に見たマスまで探しに行かせます。

追跡から捜索へ状態を切り替える
→ 07

敵数・視界・速さを面ごとに増やす
