DEEP DIVE / ENEMY MODE MACHINE

敵の「行き先」を変えると、同じ移動でも性格が変わる。

敵はmodeとtargetを持ちます。PATROLは巡回表の次のタイル、CHASEは現在のプレイヤー、SEARCHは最後に見たタイルを目標にします。距離と視界がmodeを切り替えます。

1

PATROLは決めた道を巡る

patrolRouteに角のタイルを順番で保存します。目標へ着いたらindexを1進め、最後なら0へ戻します。

target = patrolRoute[patrolIndex]
2

直線の視界でCHASEへ

xかyが同じで、その間に#がなく、距離7以内なら見えています。見えた瞬間にplayerを目標へします。

if lineOfSight && distance <= 7 { mode = chase }
3

見失ったらSEARCH

発見中はlastSeenを更新します。角で視界が切れたらそこへ向かい、着くか時間切れで巡回へ戻ります。

target = lastSeen

IN THE REAL GAME

状態遷移と移動を分ける

先にupdateModeで状態と目標を決め、その後moveEnemyがその目標へ1マス進みます。色表示も同じmodeを読みます。

visible := hasLineOfSight(enemy, player) &&
           manhattan(enemy, player) <= 7
if visible {
    mode = chase
    lastSeen = player
} else if mode == chase {
    mode = search
}

switch mode {
case patrol: target = patrolRoute[index]
case chase:  target = player
case search: target = lastSeen
}

WHY LAST SEEN?

壁越しに正解を知らない

視界が切れた敵が現在位置を追い続けないため、角へ逃げる攻略が生まれます。

WHY COLORS AND LABELS?

状態を確かめられる

色だけでなくPATROLなどの文字も出し、どの条件で変わったか遊びながら観察できます。
