I

テトリス型ゲームを作る

落ちものブロックパズル

テトリスのような、マス目を落ちてくるブロックを積んで並べるパズルを作ります。マス目を落ちる1個のブロックから、回転、固定、ライン消去、ホールドまで広げます。

はじめて読む人へ

このコースの組み立て図

一マス→形→回転→固定→行消去の順に、海老・天次郎（えび・てんじろう）と一部品ずつ作ります。

このページの1本
説明と次のコードを対応させてから、ゲームで変化を確かめよう。

func (g *Game) Update() error { return nil }

共通基礎の続きです。LEVEL 01 の Update / Draw の上に、このジャンルの部品を1ステップずつ足します。主人公が登場するステップではオリジナルの 海老・天次郎（えび・てんじろう） を使います。

LEARNING PATH

小さなルールを
つないで完成へ。

各ステップを単独で遊べる大きさにし、アルゴリズムを目で確認してから次の仕組みへ進みます。

01

1マスを落とす

一定時間ごとに1マス下へ進み、床に着いたら盤面へ固定します。

一定間隔更新とグリッド移動
→ 02

4マスの形

形の中心を基準にした位置（相対座標）で複数のマスを定義し、ひとつの物体として動かします。

中心を基準にした位置（相対座標）と形データ
→ 03

回転と壁ぎわ補正

ピースを中心のまわりで90度スピンさせ、壁にぶつかるときは少しずらした候補を試します。

中心のまわりでスピンする計算（回転行列）と回転候補
→ 04

固定とライン消去

落下中の形を盤面へ書き込み、すきまなく埋まった行を消して、残りの行を下へ詰めます（行の圧縮）。

盤面への合成と、消した分を詰める処理（行の圧縮）
→ 05

バッグ・ホールド・ゴースト

形の偏りを防ぐバッグ方式、形の保留、着地位置の予告を加えます。

公平な抽選と着地点予測
→ 06

ライン消去をショーにする

そろった行をすぐ消さず、光る中間フレーム、粒子、画面揺れを順番に見せます。

ルールの答えと見せる時間を分ける
→ 07

3つのルール海域を作る

落下速度、目標ライン、最初のおじゃま段をデータにして、性格の違う3ステージを航海します。

難しさをコードではなくステージデータにする
→ 08

Ebi Blocks

レベルごとの速度、同時消去数、連続消しを統合して完成させます。

落下速度・連続消し・スコア
