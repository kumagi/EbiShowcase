DEEP DIVE / DROP TIMER

毎フレーム動かさず、「時計」が満ちた時だけ落とす。

EbitengineのUpdateは1秒におよそ60回呼ばれます。でもブロックを毎回1マス落とすと一瞬で床へ着きます。そこで何回待ったかをtimerで数え、決めた回数で1マスだけ進めます。

1

待った回数を数える

Updateのたびにtimerへ1を足します。画面下の黄色い棒は、次の落下までの残り時間を見える形にしたものです。

timer++
2

マス単位で進む

timerが38になった時だけ行番号yへ1を足し、timerを0へ戻します。座標は小数ではなく盤面の行と列です。

if timer >= 38 { y++; timer = 0 }
3

進めないなら固定

次のマスが床か置いたブロックなら、動いているマスをboard[y][x]へ書き込み、新しいマスを上から出します。

board[y][x] = true; spawn()

TRY IT / UPDATE CLOCK

「1フレーム」で時計の目盛りを追加し、「落下判定」で先頭の目盛りを処理します。実際のゲームでは38個たまった時が落下の合図です。

IN THE REAL GAME

移動できるかを先に聞く

配列の外や使用中のマスへ入らないよう、次の行を調べてから落下か固定を選びます。

timer++
if timer >= dropFrames {
    timer = 0
    if !blocked(x, y+1) {
        y++
    } else {
        board[y][x] = true
        spawn()
    }
}

WHY A TIMER?

速さを数字で変えられる

dropFramesを小さくすれば、レベルが上がるほど速く落ちるルールも作れます。

WHY A GRID?

ぴったり積み上がる

行と列で管理すれば、ブロックが半端な位置で止まりません。
