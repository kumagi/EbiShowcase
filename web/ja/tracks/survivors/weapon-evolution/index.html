武器進化ラボ

10体倒して青いコアを拾い、20体でEbi Stormへ進化。50体撃破を目指します。

● LIVE / GO + WASMSTEP 05

HOW IT WORKS

武器を「名前と数字のセット」にする

連射間隔、弾速、弾数をWeaponデータにまとめます。必要な武器レベルとアイテムがそろったら、Ebi NeedleのデータをEbi Stormのデータへ交換します。

type Weapon struct { Name string; Cooldown, Shots int }
if kills >= 20 && hasCore {
    weapon = Weapon{Name: "Ebi Storm", Cooldown: 16, Shots: 3}
}
