Vampire Survivors型ゲームを作る

大量敵サバイバル

Vampire Survivorsのような、大量の敵から逃げながら生き延びるゲームを作ります。自動攻撃から始め、数百体の敵、経験値、強化選択、武器の進化へ発展します。

共通基礎の続きです。LEVEL 01 の Update / Draw の上に、このジャンルの部品を1ステップずつ足します。主人公はオリジナルの 海老・天次郎（えび・てんじろう） です。

SPECIALIZATION / 好きなジャンルから開始

VFXは必要な時に戻る。
まず大量敵サバイバルのルールを作る。

新しい課題は大量エンティティ、空間探索、自動攻撃、経験値と強化選択です。
下の3つは演出で迷った時の参考ページです。未受講でも、このコースから始められます。

A08イベント列と演出予算A11ヒットリアクション時間軸A06削除前スナップショット

MOTION BASELINE

動いて見えることも完成条件

攻撃イベントを列で配り、敵死亡は即削除＋表示スナップショット、被弾は段階的リアクションとして扱います。

TEST SEAM

派手さを外してもルールは同じ

敵数が増えてもイベント一回性と粒子上限が守られ、演出品質で撃破数が変わらないこと。

はじめて読む人へ

このコースの組み立て図

移動→敵→自動攻撃→経験値→強化の順に、海老・天次郎（えび・てんじろう）と一部品ずつ作ります。 このページの1本はコース全体で共通するgame状態の境界です。ルールはUpdateへ足し、Drawの順番や見た目は自由に差し替えられます。下の01から順に、その中へ各仕組みを実装していきます。

LEARNING PATH

小さな完成を
積み重ねる。

各ステップは単独で遊べるミニゲームです。前の仕組みを再利用しながら、最後に1本の作品へ統合します。

01

アリーナ回避

斜め方向でも速くなりすぎない移動を作り、敵との距離を保ちます。

斜めでも同じ速さになる移動（正規化）
→ 02

自動砲台サバイバル

一定時間ごとに最も近い敵を探し、自動で弾を発射します。

最も近い敵とクールダウン
→ 03

100体の群れ

大量の敵を再利用しながら、近くにいる敵だけを調べることで処理を軽くします。

敵の再利用と、近くだけ調べる工夫（空間分割）
→ 04

経験値と三択強化

経験値を集めてレベルアップし、そのたびにランダムな強化候補を提示します。強い強化ほど出にくくします。

ドロップ・レベル・出やすさに差をつけた抽選（重み付き抽選）
→ 05

武器進化ラボ

武器の数値と挙動をデータ化し、条件を満たした武器を進化させます。

データ駆動の強化と合成
→ 06

攻撃の手応えラボ

攻撃が当たった瞬間を、点滅・粒子・画面揺れの順に重ねて伝えます。

1回の命中を3つの時間表現へ分ける
→ 07

ウェーブ監督とボス予告

時間帯ごとに敵の速さ・大きさ・出現間隔を変え、ボスの突進前には危険リングを表示します。

時間表から難しさと予告を作る
→ 08

Ebi Survivors

制限時間、敵ウェーブ、ボス、報酬を統合して1プレイを完成させます。

ウェーブ・ボス・難易度曲線
