HOW IT WORKS

時間をゲームの台本にする

経過秒数からウェーブ番号を求めます。時間が進むほど敵の出現間隔を短く、移動を速くします。35秒になったらHPを持つボスを1体だけ登場させます。

wave := elapsedSeconds / 15
spawnInterval := max(14, 42-elapsedSeconds/2)
if elapsedSeconds == 35 { spawnBoss() }
