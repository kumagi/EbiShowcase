Cookie Clicker型ゲームを作る

放置クリッカー

スマホの放置ゲーム（クッキークリッカーのような、ボタンを押すたびに数が増えるゲーム）を作ります。1クリックの加算から、自動生産、決まった法則で値上がりする価格（価格曲線）、保存、閉じている間も進む「オフライン進行」まで育てます。

共通基礎の続きです。LEVEL 01 の Update / Draw の上に、このジャンルの部品を1ステップずつ足します。主人公はオリジナルの 海老・天次郎（えび・てんじろう） です。

はじめて読む人へ

このコースの組み立て図

押して増やす→買う→自動生産→保存の順に、海老・天次郎（えび・てんじろう）と一部品ずつ作ります。 このページの1本はコース全体で共通するUpdate→Drawの骨格です。下の01から順に、その中へ各仕組みを実装していきます。

このページの1本
説明と次のコードを対応させてから、ゲームで変化を確かめよう。

func (g *Game) Update() error { return nil }
