方向、対称性、ゆっくりした粒の順で、演出に儀式らしい秩序を作ります。

DrawLightPillar(target, whiteGold)
DrawHalo(target, rings: 3)
for i := 0; i < rays; i++ {
  ang := -math.Pi/2 + float64(i)*2*math.Pi/float64(rays)
  DrawSymmetricRay(target, ang)
}
for _, mote := range blessingMotes { mote.y += fallSlowly }

WHY FROM ABOVE?

降臨の方向を作る

中心から爆発させず、上から対象へ光を降ろすと、外から授けられた力に見えます。

WHY SYMMETRY?

秩序が神聖さになる

ランダムに回転する線より、輪・十字・左右対称の形の方が紋章や儀式として読めます。
