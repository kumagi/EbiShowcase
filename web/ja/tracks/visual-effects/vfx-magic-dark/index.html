螺旋＋求心力

初期は螺旋、更新で中心へ引っ張ります。

dx, dy := cx-p.X, cy-p.Y
pull, swirl := 0.004, 0.0025 // tune independently
pullX, pullY := dx*pull, dy*pull
swirlX, swirlY := -dy*swirl, dx*swirl
p.VX += pullX + swirlX
p.VY += pullY + swirlY
DrawImage(DarkPNG) // core normal, fringe additive
