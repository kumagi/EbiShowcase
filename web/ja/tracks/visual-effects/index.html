VFX

見た目の魔法をEbitengineで作る

ビジュアルエフェクト工房

基本編で描画の文法を手で確かめ、応用編で LEVEL 01〜12 のミニゲームそれぞれに派手な演出を足します。応用編のゴールは、エフェクトをゲーム本編の配列と分けて管理することに自分で気づくこと。

共通基礎(LEVEL 01〜12)の続きです。Update / Draw のループの上に、見た目を作る道具を1つずつ足します。主人公はオリジナルの海老・天次郎です。

LEARNING PATH

基本編と応用編。

まず LIVE GO で道具を覚え、次に12本のコアゲームへ演出を足しながら「play と fx を分ける」構造にたどり着きます。
