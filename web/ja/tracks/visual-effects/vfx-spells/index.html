魔法は
何でできている？

炎＝炎のスプライトを加算合成で上へ。水＝水滴スプライトを半透明＋重力で落とす。雷＝稲妻スプライトを数フレームだけ＋閃光＋粒。同じ道具でも、画像・向き・色・合成・寿命を変えるだけで、まったく違う魔法になります。ColorScaleはスプライトの色に掛け算するつまみ。BlendLighter（加算合成）は「色を足すと明るくなる」光の混色で、炎が背景に溶ける感じを出します。（画面のキャラクターはオリジナルの海老・天次郎。当たり判定はいつも通り簡単な形です。）

IN THE REAL GAME

同じ文法で3つの魔法

合成・色・重力を切り替えるだけで、炎・水・雷を作り分けます。

// DrawImageOptions(op) = 「どう描くか」の設定袋
// ColorScale = 画像の色に掛け算（炎ならオレンジを掛ける）
// BlendLighter = 加算合成。色を足すと明るくなる（光の混色）
op.Blend = ebiten.BlendLighter
op.ColorScale.ScaleWithColor(fire)

// 水: 半透明 + 下向き重力
op.ColorScale.ScaleAlpha(0.7)
p.vy += 0.22

// 雷: 上から下へ、左右にランダムにずらした稲妻スプライトを短寿命で並べる
y, x := top, center
for y < ground {
    bolts = append(bolts, Bolt{X: x, Y: y, Life: 16})
    y += 30
    x += rand(-25, 25) // ここでジグザグになる
}
