      IN THE DEMO

      Update を二段に分ける

      ルールと演出を同じ関数にベタ書きしない。

      func (g *Game) Update() error {
  g.updatePlay() // score, timer, targets
  g.fx.Update()  // particles live here
  return nil
}

g.fx.Burst(x, y, n, speed, tint, true)

      押した瞬間だけ粒を出すにはJustPressedを使います。isActivatedは同じ的から二度ごほうびを出さないための印です。 Go用語メモ：整数型（int）は小数を持たない数、浮動小数点型（float32/float64）は小数を持てる数、typeはデータの箱の種類を決める言葉、funcは処理をひとまとめにする関数の印です。[]TはTを順番に並べるリスト、appendはその末尾へ一つ足す命令です。Updateは数字と状態を進め、Drawはその結果を絵にします。

      海老・天次郎の見方
      Updateが天次郎の数字を変え、Drawが次の一コマを描きます。

      if justPressed && !isActivated { isActivated=true; fx.Burst() }
