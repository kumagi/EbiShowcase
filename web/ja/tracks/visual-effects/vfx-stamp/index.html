IN THE REAL GAME

同じ絵を好きな場所へ

オプションを作って Translate で位置を決め、その設定で描くだけです。

op := &ebiten.DrawImageOptions{}
op.GeoM.Translate(x, y) // 左上を (x, y) へ
screen.DrawImage(tenjiroh, op)

GeoM は絵の置き場所を覚える道具です。Translate(x,y) の x は右、y は下へ動かす量。& は設定箱そのものを渡す印です。 Go用語メモ：整数型（int）は小数を持たない数、浮動小数点型（float32/float64）は小数を持てる数、typeはデータの箱の種類を決める言葉、funcは処理をひとまとめにする関数の印です。[]TはTを順番に並べるリスト、appendはその末尾へ一つ足す命令です。Updateは数字と状態を進め、Drawはその結果を絵にします。

海老・天次郎の見方
Updateが天次郎の数字を変え、Drawが次の一コマを描きます。

op.GeoM.Translate(x, y)

func (g *Game)の(g *Game)は、この仕事がGameの値を読むための名札（レシーバー）です。
