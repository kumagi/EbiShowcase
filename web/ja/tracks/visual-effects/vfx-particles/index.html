DEEP DIVE / PARTICLES

煙や火花は
どう作る？

1個の粒はとても単純：位置と速度と寿命を持つだけ。それを配列(スライス)にたくさん持ち、Update で動かし、寿命が0になったら消します。加算合成と拡大縮小を混ぜれば、火花にも煙にもなります。

1

生成

タップした場所から粒を撒きます。

append(ps, p)
2

更新

位置を進め、重力を足します。

x+=vx; vy+=g
3

退場

寿命が尽きた粒を消します。

life--

IN THE REAL GAME

粒の一生

配列をまわして位置を進め、重力を足し、寿命で薄くして消します。

type particle struct{ x, y, vx, vy, life float64 }

for i := range ps { // Update
    ps[i].x += ps[i].vx
    ps[i].y += ps[i].vy
    ps[i].vy += gravity
    ps[i].life--
}
