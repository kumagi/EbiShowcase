GO + EBITENGINE

同じ timer を2役で使い回す

点火中は「爆発までの残り」。0になったら blasting=true にして、同じフィールドへ爆風の寿命を入れ直します。変数は1つでも、状態フラグで意味が切り替わる点に注意。

for i := range bombs {
    bombs[i].timer--
    if !bombs[i].blasting && bombs[i].timer <= 0 {
        bombs[i].blasting = true
        bombs[i].timer = blastLife
    }
}
