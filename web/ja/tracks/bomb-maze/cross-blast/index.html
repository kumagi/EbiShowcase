中心セルを追加

GO + EBITENGINE

方向と距離の二重ループにする

breakが止めるのは内側の距離ループだけです。そのため右が壁で止まっても、次の上・下・左は調べられます。

cells := []point{center}
for _, direction := range rays {
    for step := 1; step <= blastRange; step++ {
        next := point{
            center.x + direction.x*step,
            center.y + direction.y*step,
        }
        if hardWall(next) {
            break
        }
        cells = append(cells, next)
    }
}

なぜセルの配列にする？

描画、プレイヤーの自爆判定、ビーコンの命中判定が、すべて同じblast配列を使えるからです。

壁の向こうへ届かない理由

壁を見つけた瞬間に追加前で止めます。「壁も爆風に含めるか」は順番を変えるだけで設計できます。

チャレンジ

射程を2や4に変え、予定セルの数と逃げる難しさがどう変わるか比べてみよう。
