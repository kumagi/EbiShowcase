ボンバーマン型の爆風ゲームを作る

爆弾迷路アクション

ボンバーマンのような、迷路に爆弾を置いて壁や敵を吹き飛ばすゲームを作ります。時限爆弾から、上下左右に壁で止まるまで伸びていく十字の爆風、壊れる壁、連鎖爆発、アイテム、敵AIへ進みます。

共通基礎の続きです。LEVEL 01 の Update / Draw の上に、このジャンルの部品を1ステップずつ足します。主人公が登場するステップではオリジナルの 海老・天次郎（えび・てんじろう） を使います。

LEARNING PATH

ひとつの仕組みから、
遊べる世界へ。

データの変化とアルゴリズムを小さなゲームで確かめ、前のステップを再利用しながら完成形へ進みます。

時限爆弾を置く

立っているマスへ爆弾を置き、一定時間後に状態を爆発へ変えます。

十字に伸びる爆風

爆弾を置いたマスから上・下・左・右へ、爆風が1マスずつ伸びていくように調べ、固い壁に当たるか届く距離（射程）まで来たら止めます。

上下左右に伸びていく爆風の調べ方（4方向レイ探索）
→ 03

壊れる壁とアイテム

爆風を受けた壁を床へ変え、確率で能力アップを出現させます。

タイル変化と確率ドロップ
→ 04

爆弾の連鎖爆発

爆風が別の爆弾へ届いたら即座に爆発させ、その爆発が伝わっていきます（イベント伝播）。同じ爆弾を二度爆発させないよう記録しておきます（二重処理防止）。

爆発が次々伝わる仕組み（イベント伝播）と二重処理防止
