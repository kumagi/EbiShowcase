UPDATE と DRAW の境界

1

Update が担当

キー・タッチを読み、位置・HP・ゲージ・タイマー・盤面を変えます。判定やキュー操作もこちらです。

入力 → ルール → game を更新
2

Draw が担当

できあがった game の値を読んで、絵・文字・エフェクトを置きます。game の値は変えません。

game を読む → ピクセルへ投影

このページのルールは Update 側（または Update が呼ぶ純粋な関数）へ置き、Draw はその結果を表示するだけにします。
