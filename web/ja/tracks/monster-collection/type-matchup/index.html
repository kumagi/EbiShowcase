DEEP DIVE / MATCHUP TABLE

if文を増やさず、2つのタイプで表を引く。

攻撃タイプを行、防御タイプを列にした3×3の倍率表を用意します。技の威力に倍率をかければ、同じダメージ式で「強い」「普通」「弱い」を計算できます。

行＝攻撃 / 列＝防御
TIDE守FLAME守LEAF守
TIDE攻120.5
FLAME攻0.512
LEAF攻20.51

例: 炎で葉を殴る → 行FLAME・列LEAF → 表の右端の 2

1

タイプを番号にする

TIDE=0、FLAME=1、LEAF=2。element（属性）という名前で番号型にします。この番号を配列の行と列へそのまま使えます。

type element int
2

攻撃行×防御列を引く

[3][3]float64 は「3行の表の、各行に3列」です。最初の番号＝攻撃（行）、次の番号＝防御（列）。TIDE攻撃×FLAME防御なら matchup[0][1] → 2.0。

mult := matchup[attack][defense]
3

同じ表を反撃にも使う

敵の反撃タイプを攻撃行、自分が選んだ構えを防御列にします。攻撃だけ強くても反撃が痛い場合があります。

counterMult := matchup[counter][chosen]

IN THE REAL GAME

倍率表はデータとして1か所に置く

ダメージ計算は表の中身を知らず、返された倍率を使うだけです。相性調整は表の数字だけを変えます。

var matchup = [3][3]float64{
    {1,   2, 0.5}, // Tide attack
    {0.5, 1,   2}, // Flame attack
    {2, 0.5,   1}, // Leaf attack
}
mult := matchup[chosen][enemyDefense]
damage := int(basePower * mult)
