ぷよぷよ型の連鎖を作る

落下ペア連鎖パズル

ぷよぷよのような、同じ色を4つ以上つなげて消す連鎖パズルを作ります。2個組の落下から、同色の連結探索、消去、重力、再探索、連鎖得点へ進みます。

共通基礎の続きです。LEVEL 01 の Update / Draw の上に、このジャンルの部品を1ステップずつ足します。主人公が登場するステップではオリジナルの 海老・天次郎（えび・てんじろう） を使います。

はじめて読む人へ

このコースの組み立て図

二個組→回転→同色探索→落下→連鎖の順に、海老・天次郎（えび・てんじろう）と一部品ずつ作ります。

海老・天次郎の見方
Updateが天次郎の数字を変え、Drawが次の一コマを描きます。

func (g *Game) Update() error { return nil }

func (g *Game)の(g *Game)は、この仕事がGameの値を読むための名札（レシーバー）です。

遊べる世界へ。

データの変化とアルゴリズムを小さなゲームで確かめ、前のステップを再利用しながら完成形へ進みます。

ペアを回転する

同じ色の仲間探し

上下左右につながる同じ色を塗り広げるように集め（Flood Fill）、同じマスを2度数えないよう調べ済みを記録しながら、4個以上か調べます。関数が自分自身を呼ぶ「再帰」という書き方を使います。

同色の塊を塗り広げて見つける（Flood Fill）と調べ済みの記録
→ 04

消去とちぎれ落ち

条件を満たす全グループを同時に消し、残ったピースを下へ落とします。

複数グループ消去と列圧縮
→ 05

連鎖スコア研究所

まず消える場所が見えてから連鎖が起きるように、落下後にもう一度探索し、続けて消えた回数が多いほど得点が上がる倍率（連鎖係数）をかけます。

再探索と、連鎖が続くほど上がる得点倍率（連鎖係数）
→ 06

連鎖をアニメーションで伝える

予告・消去・落下・次の消去に時間を置き、連鎖が大きいほど粒子と揺れを強くします。

盤面の変化を読める時間へ分ける
→ 07

3つの海底デュエルを設計する

勝利に必要なおじゃま数、ミス上限、反撃量をステージデータにして3連戦を作ります。

同じ連鎖ルールへ違う対戦条件を渡す
