DEEP DIVE / CLEAR THEN COMPACT

「消す」と「落とす」を、同じループに混ぜない。

前のSTEPで見つけた4個以上のグループをmarkedへ全部集めます。まずmarkedをすべて空へ変え、盤面の穴を確定します。その後にだけ、各列を下から読み直して圧縮します。

1

消す対象を先に全部集める

盤面を書き換える前にFlood Fillを終え、条件を通った複数グループの座標を一つのmarkedへ入れます。

marked[p] = true
2

全グループを同時に空へ

markedを一周してemptyを書きます。探索中に消さないので、後のグループのつながりを壊しません。

for p := range marked { board[p.y][p.x] = empty }
3

列を下から圧縮

writeを最下段に置き、下から上へ空でないセルだけコピーします。最後にwriteより上をemptyで埋めます。

board[write][x] = board[read][x]; write--

IN THE REAL GAME

readとwriteで空きを飛ばす

readは全セルを見ます。writeは次に詰める場所です。空セルではwriteを動かさないため、残った色だけが下へ集まります。

for x := 0; x < cols; x++ {
    write := rows - 1
    for read := rows-1; read >= 0; read-- {
        if board[read][x] == empty { continue }
        board[write][x] = board[read][x]
        write--
    }
    for write >= 0 {
        board[write][x] = empty
        write--
    }
}

WHY SIMULTANEOUS?

探索結果を途中で変えない

一組ずつ消すと、その後に調べる組のつながりが変わります。全対象を決めてから一斉に変えます。

WHY PER COLUMN?

横位置を守る

重力は下方向だけなので、列ごとに処理すればxを変えずに穴だけ詰められます。
