LEARNING PATH

一歩ずつ、冒険を組み立てる。

8方向移動と剣の一振りから始め、部屋・鍵・道具の仕掛け・多段階ボスを足して、4部屋のEbiオリジナルダンジョンへ進みます。

01

8方向に同じ速さで歩く

上下左右を組み合わせ、斜めだけ速くならないように方向ベクトルの長さを1へそろえます。

斜めの長さをNormalizeで1にする
→ 02

向いている先に剣を出す

主人公の位置と向きから攻撃矩形を作り、前にいる敵だけへダメージを与えます。

faceから攻撃矩形の位置を決める
→ 03

ノックバックと無敵時間

敵に触れると後ろへ押されて点滅し、短い無敵時間のあとで再びダメージを受けます。

被弾したら無敵カウントを始める
→ 04

全滅すると開く3つの部屋

部屋へ入ると出口を閉じ、敵の残りが0になった時だけ次の部屋へ進めます。

敵の残数を部屋状態で数える
→ 05

鍵・扉・宝箱をつなぐ

鍵を拾う、対応する扉を開ける、奥の宝箱へ届くという状態の受け渡しを作ります。

鍵を消費してdoorOpenを変える
→ 06

3つの道具で仕掛けを解く

剣・風・ランプを切り替え、ツタ・風車・影の印に正しい道具を使います。

仕掛けに必要なkindを持たせる
→ 07

HPで変わる3段階ボス

ボスのHPが減ると防御・突進・嵐へ行動が変わり、色と状態名で次の危険を知らせます。

段階ごとに速度と攻撃間隔を変える
→ 08

Ebi Relic Dungeon

鍵の部屋、全滅戦、3道具の仕掛け、3段階ボスを一つの4部屋ダンジョンとして攻略します。

攻撃・被弾・解錠・仕掛けを同じ更新順で解決する
