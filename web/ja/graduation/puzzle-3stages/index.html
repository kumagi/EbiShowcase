GRADUATION 03

データを増やして、パズルを増やす。

ルールをコピーせず、ステージ名と目標回数を配列へ追加する3ステージパズルです。

0. DOWNLOAD

空の Go フォルダで始める

starter の3ファイルを Raw 表示から保存します。go mod init example.com/three-puzzles、go get github.com/hajimehoshi/ebiten/v2、go test の順で実行します。clone は必要ありません。

1. START

最初のデータを選ぶ

TestFreshProgressStartsAtFirstDataStage は TODO 1。Stage 0 が Stages[0] を指すように、空の Progress を作ります。

2. ADVANCE

目標回数をデータから読む

TestPuzzleTargetAdvancesOneStage と TestLastDataStageClearsInsteadOfOverflowing は TODO 3。Presses を数え、現在の Stages[p.Stage].Target に届いたら一段だけ進めます。最後は配列を越えず Clear にします。

3. RESET

進行を最初へ戻す

TestRestartMakesFreshProgress は TODO 2。古いステージ、入力回数、Clear を残さず NewProgress() を使います。
