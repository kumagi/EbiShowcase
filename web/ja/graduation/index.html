学んだ部品を、自分のゲームにする。

3つの小さな完成ゲームを、赤いテストから作ります。遊べるだけでなく、ルールを自分で変更して確かめる卒業制作です。

BEFORE YOU START

用意するもの

Go とブラウザだけです。リポジトリを clone する必要はありません。各課題ページから starter の3ファイルを Raw 表示で保存し、空のフォルダで始めます。

mkdir my-ebi-game && cd my-ebi-game; go mod init example.com/my-ebi-game; go get github.com/hajimehoshi/ebiten/v2; go test

最初の go test が赤くなるのは成功です。テスト名に対応した TODO を一つずつ埋め、緑になってから go run . で遊びます。
