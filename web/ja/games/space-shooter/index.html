HOW IT WORKS

同じ種類を「スライス」にまとめる

画面に弾が100個あっても、1個ずつ名前を付ける必要はありません。弾の位置と速さをスライスへ追加し、Updateで全員を少しずつ動かします。画面外へ出た弾や敵に当たった弾は取り除きます。

for i := range bullets {
    bullets[i].Y += bullets[i].VY
}
bullets = removeOutside(bullets)
