HOW IT WORKS

角度を等分して弾の速度に変える

円を弾の数で等分し、cosで横速度、sinで縦速度を作ると円形弾幕になります。開始角度を毎回ずらすだけで、回転して見える模様になります。

angle := spin + i*2*math.Pi/count
vx := math.Cos(angle) * speed
vy := math.Sin(angle) * speed
