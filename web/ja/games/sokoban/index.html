HOW IT WORKS

絵ではなく、マスの意味を読む

ステージは文字の表として保存できます。#は壁、$は箱、.はゴールです。移動先が箱なら、その1マス先まで空いている時だけ箱と人を動かします。

next := player.Add(direction)
if boxAt(next) && isEmpty(next.Add(direction)) {
    moveBox(next, direction)
}
movePlayer(next)
