HOW IT WORKS

敵の「いま」を状態で表す

敵には巡回中と追跡中という状態があります。プレイヤーが近づくと追跡へ、遠ざかると巡回へ戻します。状態ごとにUpdateの行動を切り替えると、考えているような敵になります。

switch enemy.State {
case Patrol: enemy.walkRoute()
case Chase:  enemy.moveToward(player)
}

GAME + LESSON

敵AIを小さな判断に分ける

01

感じる

敵からプレイヤーまでの距離を測ります。

02

決める

距離を条件に、巡回と追跡の状態を切り替えます。

03

行動する

選ばれた状態に対応する移動を実行します。
