HOW IT WORKS

横と縦を分けて、めり込みを戻す

まずXだけ動かして壁と重なった分を戻し、次にYだけ動かして床との重なりを戻します。カメラ座標を引いて描けば、キャラクターを追うスクロールになります。

player.X += vx
resolveHorizontal()
player.Y += vy
resolveVertical()
screenX := player.X - cameraX

GAME + LESSON

マリオ型アクションの土台

01

加速する

左右入力を速度へ足し、重力を毎フレーム縦速度へ足します。

02

押し戻す

地形との重なりを移動方向と反対へ戻して、床や壁で止めます。

03

世界を映す

プレイヤーを追うカメラを使い、画面より広いコースを描きます。
