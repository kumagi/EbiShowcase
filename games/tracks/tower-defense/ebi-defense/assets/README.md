were generated using Codex's built-in image generation tool on 2026-07-16.

- `pearl-gate-battlefield.png`: premium-mobile underwater reef battlefield,
  invasion rift at left, protected palace at right, and readable build terraces.
- `defense-towers.png`: bolt, slow, and splash tower atlas; the flat `#ff00ff`
  chroma background was removed locally.
- `defense-battle.png`: runner, armored raider, boss, pearl sanctuary, and three
  distinct projectile effects; the flat chroma background was removed locally.

Prompts required original all-ages contemporary Japanese mobile-game art,
distinct tactical silhouettes, no text/logo/watermark, and no copyrighted
character resemblance.
