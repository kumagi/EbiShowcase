# Ebi Maze generated art

These original assets are licensed under Apache-2.0 with this repository. They
were generated with Codex's built-in image generation tool on 2026-07-16.

- `pearl-labyrinth.png`: premium-mobile submerged palace floor, with a calm
  sapphire maze area and ornate shell, brass, coral, and moonlight framing.
- `maze-cast.png`: five-column atlas containing the pearl diver player, teal
  patrol guard, scarlet chase sentinel, violet search scout, and pearl goal.
  Its flat `#ff00ff` chroma background was removed locally.

Prompts required original all-ages designs, contemporary Japanese mobile-game
rendering, distinct role silhouettes, no text/logo/watermark, and no copyrighted
character resemblance.
